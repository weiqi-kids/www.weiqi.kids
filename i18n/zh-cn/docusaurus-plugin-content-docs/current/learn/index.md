# 给围棋棋友

欢迎来到围棋棋友专区！无论你是刚接触围棋的新手，还是想要精进棋艺的爱好者，这里都有适合你的学习资源。

## 本区块适合谁

- **完全新手**：从未接触过围棋，想从零开始学习
- **初学者**：已经知道基本规则，想要建立正确的观念
- **业余棋友**：有一定基础，希望系统性地提升棋力
- **回归棋友**：曾经学过围棋，想要重新拾起这项兴趣

## 学习路径建议

### 阶段一：打好基础（1-3 个月）

1. 熟读规则，确保理解「气」与「眼」的概念
2. 学习基本术语，能够看懂棋谱解说
3. 在 9 路或 13 路小棋盘上练习
4. 养成良好的对弈礼仪

### 阶段二：建立棋感（3-6 个月）

1. 理解开局的大方向
2. 练习简单的死活题
3. 开始在 19 路大棋盘下棋
4. 每盘棋后简单复盘

### 阶段三：系统提升（6 个月以上）

1. 学习基本定式与常见变化
2. 培养形势判断能力
3. 善用 AI 工具分析自己的棋局
4. 欣赏职业棋士的对局，培养大局观

:::tip 学习建议
围棋的进步需要时间和耐心。比起急着学很多知识，不如先把基础打稳。每天下一盘棋、做几道死活题，持之以恒比什么都重要。
:::
