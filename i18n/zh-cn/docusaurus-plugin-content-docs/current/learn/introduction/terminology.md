# 围棋术语

学习围棋术语，能帮助你看懂棋谱、理解解说，并与其他棋友交流。

## 棋盘术语

### 棋盘的重要位置

| 术语 | 位置 | 说明 |
|------|------|------|
| **天元** | 棋盘正中央 | 19 路棋盘的第 10 线交点（10,10） |
| **星位** | 四个角落附近的黑点 | 距离边线各 4 条线的位置（4,4） |
| **小目** | 角落的常见落子点 | (3,4) 或 (4,3) 的位置 |
| **三三** | 角落最深处 | (3,3) 的位置，直接取角的下法 |
| **目外** | 角落较高的位置 | (3,5) 或 (5,3) 的位置 |
| **高目** | 角落的高位 | (4,5) 或 (5,4) 的位置 |

### 棋盘的区域

| 术语 | 说明 |
|------|------|
| **角** | 棋盘四个角落区域，通常指 4 线以内 |
| **边** | 棋盘四边区域，角落之间的部分 |
| **中腹** | 棋盘中央区域，也称「中央」或「腹地」 |

### 位置的高低

| 术语 | 说明 |
|------|------|
| **一线** | 最外面的边线（死亡线） |
| **二线** | 第二条线（败线） |
| **三线** | 第三条线（实地线） |
| **四线** | 第四条线（势力线） |

:::note 俗谚
「一线求活缘木鱼，二线出头恐不盈」—— 一、二线是不利的位置。
:::

## 下棋术语

### 基本动作

| 术语 | 说明 |
|------|------|
| **落子** | 将棋子放到棋盘上 |
| **提子** | 把没有气的对方棋子拿走 |
| **弃子** | 故意放弃某些棋子不救 |

### 攻击相关

| 术语 | 说明 |
|------|------|
| **打吃 / 叫吃** | 让对方棋子只剩一口气 |
| **双叫吃** | 同时叫吃两块棋，对方只能救一块 |
| **征子（征吃）** | 连续叫吃，对方逃不出的吃法 |
| **枷** | 宽松地封住对方，跑也会被吃 |

### 棋子的移动方式

| 术语 | 说明 | 形状 |
|------|------|------|
| **长** | 沿着直线连接出去 | 直线延伸 |
| **跳** | 隔一格跳出（同线上） | 中间空一格 |
| **飞** | 对角方向移动，中间空一格 | 类似象棋的象 |
| **大飞** | 对角方向移动，距离更远 | 比飞更远 |
| **尖** | 对角紧靠（斜向相邻） | 斜着连接 |
| **关** | 同线上隔一格跳（同跳） | 横向或纵向 |

### 接触战术语

| 术语 | 说明 |
|------|------|
| **扳** | 在对方棋子斜角顶住 |
| **断** | 切断对方棋子的连接 |
| **接** | 把自己的棋子连接起来 |
| **挡** | 挡住对方的去路 |
| **夹** | 从两边夹住对方的棋子 |
| **靠** | 紧贴着对方的棋子下 |
| **贴** | 紧挨着下，通常在对方棋子旁边 |
| **压** | 从上方压住对方，让对方变低 |
| **托** | 在对方棋子下方托住 |
| **点** | 在关键位置下一子（常在眼位或要点） |
| **刺** | 往对方的弱点下一子威胁 |

## 棋形术语

### 好形

| 术语 | 说明 |
|------|------|
| **虎** | 两子对角，中间空一点，形成虎口 |
| **双** | 两子连接后的基本形状 |
| **厚实** | 棋形没有弱点，非常稳固 |
| **轻灵** | 棋形有弹性，不怕被攻 |

### 基本概念

| 术语 | 说明 |
|------|------|
| **眼** | 被己方棋子围住的空点 |
| **气** | 棋子或棋块周围的空点（生命线） |
| **断点** | 棋子连接的弱点，可能被切断的地方 |
| **连接** | 棋子相连形成一体 |

### 不好的形

| 术语 | 说明 |
|------|------|
| **愚形** | 效率差的棋形 |
| **凝形** | 棋子挤在一起，效率不高 |
| **空三角** | 三子围成空三角，典型的愚形 |
| **裂形** | 棋形被分开，容易被攻击 |

## 对局术语

### 手的顺序

| 术语 | 说明 |
|------|------|
| **先手** | 下完这手后，轮到我方继续主导 |
| **后手** | 下完这手后，需要让对方先动 |
| **抢先** | 在局部不收完，转向其他大场 |
| **先手利** | 先手得到的便宜 |

### 对局阶段

| 术语 | 说明 |
|------|------|
| **布局** | 开局阶段，双方占领大场 |
| **中盘** | 战斗最激烈的阶段 |
| **官子** | 终局阶段，收拾边界 |
| **收官** | 进行官子，把边界收好 |

### 局面判断

| 术语 | 说明 |
|------|------|
| **优势** | 局面领先 |
| **劣势** | 局面落后 |
| **细棋** | 双方差距很小的局面 |
| **大局** | 影响整体局势的下法或判断 |

### 对局结果

| 术语 | 说明 |
|------|------|
| **黑胜 X 目** | 黑方赢 X 目 |
| **白胜 X 目** | 白方赢 X 目 |
| **和棋** | 双方目数相同（非常罕见） |
| **中盘胜** | 不计算目数的胜利（对方认输或弃权） |

## 进阶术语

这些术语在进阶学习时会经常遇到：

| 术语 | 说明 |
|------|------|
| **定式** | 开局时双方最佳应对的固定下法 |
| **手筋** | 巧妙的手法，精彩的一手 |
| **俗手** | 普通、效率不高的下法 |
| **缓着** | 不够紧迫、让对方得利的下法 |
| **急所** | 必须马上下的重要位置 |
| **大场** | 价值很大的位置 |
| **厚势** | 厚实的势力，对外有影响力 |
| **实地** | 确定围住的地盘 |
| **模样** | 尚未完成但有潜力的地盘 |

:::tip 学习建议
不需要一次记住所有术语。建议先熟悉「下棋术语」中的基本用词，其他的在实战中慢慢学习即可。
:::
