# 都市オンラインリーグ

オンライン都市対抗囲碁大会で、各地の棋士がインターネットを通じて対局し、地元の名誉をかけて競い合います！

## 大会概要

都市オンラインリーグは、好棋ベビー協会が主催するオンライン団体戦です。各都市がチームを編成し、弈客囲碁プラットフォームで対局します。

![弈客プラットフォーム](/img/activities/city-league/platform.jpg)

10都市がチームを編成して参加：

### 対局風景

![台中チーム対局中](/img/activities/city-league/taichung-team.jpg)

各都市の選手がそれぞれの会場に集まり、オンラインで同時対局。会場は熱気に包まれました。

## 大会ルール

- **対局プラットフォーム**：弈客囲碁
- **形式**：団体対抗戦
- **チーム人数**：5名
- **持ち時間**：各10分、30秒×3回の秒読み
