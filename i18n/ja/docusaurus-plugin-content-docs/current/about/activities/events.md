# 囲碁イベント

協会が主催・参加した各種囲碁イベントの記録です。

## 第1回好棋ベビーカップ親子囲碁カーニバル

**開催日**：2024年6月
**会場**：和平高校

![カーニバル会場](/img/activities/carnival/venue.jpg)

第1回好棋ベビーカップ親子囲碁カーニバルでは、囲碁大会と親子アクティビティを組み合わせ、ご家族みんなで囲碁の楽しさを体験していただきました。

### イベント内容

- グループ別囲碁大会
- 親子ふれあいゲーム
- 囲碁体験ブース
- 表彰式

## 2024年台北市体育総会全国囲碁オープン

**開催日**：2024年1月28日
**会場**：蓬萊小学校

![大会会場](/img/activities/events/taipei-tournament.jpg)

協会が共催した台北市体育総会全国囲碁オープン春季大会には、台湾各地から多くの棋士が参加しました。
