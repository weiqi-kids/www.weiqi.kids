# 好棋クラブ

毎月開催の囲碁交流会。テーマ講座、親善対局、ネットワーキングを組み合わせたイベントです。

## イベント内容

好棋クラブは毎月台中で開催。内容は以下の通りです：

- **5分間テーマ講座**：各分野の専門家を招いた講演
- **1時間の交流タイム**：深い議論と情報交換
- **親善対局**：棋士同士の対局
- **初心者レッスン**：5分間の純碁入門

## クラブの特徴

### 異業種交流

好棋クラブは単なる囲碁の集まりではなく、異業種交流の場でもあります。様々な分野の講師を招き、囲碁を楽しみながら視野を広げることができます。

### 効果的なマッチング

- 5分間のプレゼンで専門性を理解
- グループチャットで継続的な交流
- 1時間の深い対話

## 参加方法

- **親善対局**：無料
- **指導碁**：500元/人
- **イベントスポンサー**：500ポイント/人
