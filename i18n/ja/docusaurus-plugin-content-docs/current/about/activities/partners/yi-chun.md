# 謝依純

**プロの声優**

## プロフィール

謝依純はプロの声優で、好棋ベビー協会のプロモーション動画のナレーションを担当しています。広告、企業PR、アニメ吹替など幅広い分野で活躍しています。

## ナレーション実績

謝依純は多数の有名ブランドのナレーションを担当：

- 南山人寿 商品編
- 康是美 Saeve発売プロモーション
- PIXNET 企業PR
- ヒルズ ペットフードCM
- マクドナルド 冷たい飲み物CM
- 韓国ドラマ「また！？オ・ヘヨン」吹替

## 協業内容

好棋ベビー協会のプロモーション動画、イベント紹介のナレーションを担当。温かく親しみやすい声で囲碁文化の魅力を伝えています。

## サービス内容

- 企業PRナレーション
- イベント紹介ナレーション
- アニメキャラクター吹替
- CM・広告ナレーション
