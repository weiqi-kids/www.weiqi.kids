# 呉荻

**イラストレーター**

## プロフィール

呉荻はプロのイラストレーターで、好棋ベビー協会のマスコットキャラクターをデザインしました。朵拉（キツネ）、岈荹（タヌキ）、そして辰年・寅年の特別キャラクターなどがあります。

## 協業作品

### キャラクターデザイン：朵拉と岈荹

![朵拉](/img/partners/wu-di-dora.png)

**朵拉**は和服を着たキツネのキャラクター。明るく元気な性格です。

### 干支キャラクターデザイン

![龍と虎](/img/partners/wu-di-dragon.jpg)

好棋ベビーのために制作した辰年・寅年の特別キャラクター。囲碁と伝統文化の要素を融合させています。

## 作品の特徴

- 可愛らしいデフォルメスタイル
- 伝統文化とモダンな要素の融合
- 子ども向け囲碁普及に最適
