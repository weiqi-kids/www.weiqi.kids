# 毛子

**イラストレーター**

## プロフィール

毛子はイラストレーターで、好棋ベビー協会のために囲碁テーマの壁紙シリーズを制作しました。ファンタジックなスタイルで囲碁の要素を日常生活に取り入れています。

## 協業作品

### 囲碁テーマ壁紙

![囲碁テーマ壁紙](/img/partners/mao-zi-wallpaper.png)

星空と碁石をテーマにしたスマートフォン壁紙。黒白の碁石が可愛らしい惑星となり、宇宙を漂っています。

## 作品の特徴

- 幻想的で美しいイラストスタイル
- 囲碁要素と自然風景の融合
- スマートフォン・PC版を提供
- 毎月テーマを更新
