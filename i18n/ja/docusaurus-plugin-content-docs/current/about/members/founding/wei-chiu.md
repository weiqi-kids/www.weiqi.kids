# 邱尚偉 Wei

![邱尚偉](/img/members/wei-chiu.jpg)

**藍海策略営銷管理顧問股份有限公司 企画部マネージャー**

## プロフィール

邱尚偉は9年のイベント企画経験を持つプロフェッショナルで、台湾初のファイヤーダンス・ストリートパフォーマーでもあります。創世基金会斗六院でソーシャルワーカーを務めた経験があり、チャリティーイベントにも精通しています。好棋ベビー協会ではイベント運営とステージ管理を担当しています。

## 専門分野

- **イベント企画**
  - セミナー系：国際フォーラム、ワークショップ、展示会
  - スポーツ系：スポーツイベント（不老棒球聯盟）、全国大会
  - 交流会：産地直送ディナー、パーティー
- **ブランドマーケティング**：食農教育、ブランドプロモーション
- **チャリティーイベント**：バザー、O2Oイベント
- **映像制作**：撮影・編集
