# 潘天健

![潘天健](/img/members/tien-chien-pan.png)

**沁安統合医学クリニック 院長**

## プロフィール

潘天健医師は西洋医学の内科専門医資格を持つ中医師です。中医学、西洋医学、分子栄養学などを組み合わせた統合的な医療サービスを提供しています。

## 現職

- 天健国際生医股份有限公司 会長
