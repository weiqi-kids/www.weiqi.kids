# 人類の囲碁発展史

囲碁は人類史上最も古く、最も奥深い盤上ゲームの1つです。中国古代の伝説的起源から、数千年の発展を経て、囲碁は単なるゲームにとどまらず、東アジア文化の重要な構成要素となり、哲学、芸術、競技の多重的な意味を担っています。

## 発展の脈絡概観

### 古代（紀元前約2000年〜紀元1600年）

囲碁は中国で誕生し、伝説では堯帝が創ったとされ、息子の丹朱を教えるために使われました。最古の文字記録は春秋戦国時代に現れ、その後囲碁は朝鮮半島と日本に伝わり、東アジア三国共通の文化遺産となりました。

この長い歳月の中で、囲碁は宮廷貴族の娯楽から、徐々に文人が修養を積むためのツールへと発展し、専門的な棋譜記録や理論著作が現れ始めました。

### 近代（1600年〜1950年）

日本は江戸時代に独特の碁院制度を確立し、四大棋家（本因坊、井上、安井、林）の競争が囲碁理論と技術の急速な発展を推進しました。この時期、日本は世界囲碁の中心となり、多くの伝説的棋士を輩出しました。

中国の呉清源は日本で発展し、「新布石」時代を切り開き、その卓越した棋芸で数十年にわたり日本棋界に君臨しました。

### 現代（1950年〜2015年）

第二次世界大戦後、日本棋界は「六超一強」の黄金時代を迎えました。しかし1980年代から韓国囲碁が台頭し、曹薫鉉、李昌鎬、イ・セドルらが相次いで世界棋界に君臨しました。

21世紀に入り、中国囲碁が復興を始め、柯潔らの新世代棋士が台頭し、韓国と激しい競争態勢を形成しました。
