# AI時代

2016年、AlphaGoが世界チャンピオンのイ・セドルを破り、囲碁界全体に衝撃が走りました。その瞬間から、囲碁は全く新しいAI時代に突入しました。

## この時代の意義

人工知能は人間の棋士に勝っただけでなく、より重要なのは：

- **既存の認識を覆した**：多くの「常識」が覆された
- **新しい打ち方を開拓した**：人間が考えもしなかった手法を発見した
- **学習方法を変えた**：AIが強力な学習ツールとなった

## アマチュア愛好家への影響

AIの出現はアマチュア愛好家にとって、機会であり挑戦でもあります：

### 機会

- いつでもトップレベルの相手と対局できる
- 自分の棋譜を即座に分析できる
- 学習リソースがより豊富になった

### 挑戦

- AIツールをどう正しく使うか
- AIへの過度な依存を避ける
- AI時代に囲碁を打つ楽しさを保つ
