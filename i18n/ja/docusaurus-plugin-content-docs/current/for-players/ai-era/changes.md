# AIがもたらした変化

AlphaGoの出現は人間のトップ棋士に勝っただけでなく、より重要なのは囲碁に対する私たちの理解を変えたことです。過去「常識」と考えられていた多くの観念が、AIの分析により再検討が必要になりました。

## 主な棋風の変化

### 三々点が一般的に

**過去の観念**：序盤で直接相手の星に三々を打つと、相手に強力な外勢を与え、「損」な打ち方とされていました。

**AIの見方**：三々点は損ではありません。相手に外勢を与えても、自分が得る実利は非常に大きく、しかも外勢の価値は以前考えられていたほど高くありません。

**現在のトレンド**：
- 序盤で三々を打つことが非常に一般的に
- プロ棋士は5〜10手目で三々を打つことが多い
- 双方が直接三々に打つ序盤も出現

### 実利をより重視

**過去の観念**：「外勢」は非常に価値があり、相手を攻撃したり、より多くの地を囲うのに使えるとされていました。

**AIの分析**：外勢の実際の価値はしばしば過大評価されています。実利は確定した利益であり、外勢が地に転化できるかは多くの不確定要素があります。

**アマチュア愛好家への示唆**：
- 「厚み」を過信しない
- 実利の価値は想像以上に重要
- 外勢が本当に機能するかを判断することを学ぶ

### 序盤の着点の変化

| 位置 | AI時代以前 | AI時代以降 |
|------|-------------|-------------|
| 星（4,4） | 最も一般的な序盤 | 依然として一般的だが比率は低下 |
| 小目（3,4） | 非常に一般的 | 依然として主流の選択 |
| 三々（3,3） | 直接使用は少なかった | 非常に人気に |
| 高目（4,5） | 時々使用 | 使用頻度が増加 |

## 「常識」への挑戦

AIは私たちが当然と思っていた多くの囲碁の観念に挑戦しました：

### 覆された「常識」

| 過去の常識 | AIの見方 |
|------------|-----------|
| 序盤で三々を打つな | 三々点は完全に可能 |
| 外勢は非常に価値がある | 外勢は過大評価されやすい |
| ある定石が「最善の打ち方」 | 多くの新変化を発見 |
| 二線は「敗線」 | 時に二線がむしろ良い点 |

### 依然として成り立つ原則

多くの観念が覆されましたが、いくつかの基本原則は依然として正しいです：

- **基本の棋理**：気、眼、連結などの基本概念は変わらない
- **効率の原則**：金隅銀辺草腹は大体成り立つ
- **石の形の原則**：愚形は依然として愚形
- **死活判断**：これは客観的であり、AIで変わらない

:::note 重要な区別
AIが変えたのは主に「価値判断」（何がより良いか）であり、「ルール的な知識」（何が正しいか）ではありません。
:::

## アマチュア愛好家の適応方法

### 1. AIの打ち方をすべて受け入れる必要はない

AIの碁は超強力な計算力の上に成り立っています。AIの多くの打ち方は：

- 精密な後続計算が必要
- 人間の棋力では扱いにくい可能性
- 直接真似するのに適していない

**アドバイス**：AIの打ち方の背後にある「概念」を理解し、具体的な手順を丸暗記しないこと。

### 2. 自分の観念を再検討する

AIが私たちにくれた最大の贈り物は：**疑問を持つことを学ぶ**こと。

- この打ち方は本当に良くないのか？それとも「常識」がそう言っているだけか？
- この外勢は本当にそれほどの価値があるのか？
- 自分の判断に盲点はないか？

### 3. 開かれた心を持つ

| 閉じた態度 | 開かれた態度 |
|------------|------------|
| 「この打ち方は絶対に違う」 | 「なぜAIがこう打つのか考えてみよう」 |
| 「先生が教えたことは絶対に正しい」 | 「先生の経験は貴重だが、議論もできる」 |
| 「AIの碁は人間に向かない」 | 「どんな概念を学べるか見てみよう」 |

### 4. 自分に合った上達方法に集中する

アマチュア愛好家にとって、最も重要なのはAIの新しい打ち方を学ぶことではなく：

- 基本をしっかり固める
- 明らかなミスを減らす
- 局面に対する感覚を養う
- 対局のプロセスを楽しむ

## AI時代の棋譜研究

### AIが推奨する打ち方の見方

AIで自分の棋譜を分析する時：

1. **「最善手」だけを見ない**：なぜその手が良いのかを理解する
2. **勝率変化に注目**：どこが転換点かを見る
3. **大きな方向性に注目**：細かい差異にこだわりすぎない

### 「勝率」の意味を理解する

AIが表示する勝率（例：黒55%）は：

- AIの棋力での黒の勝率
- 双方が最善手を打つという仮定に基づく
- 人間棋士にとっては参考程度

:::caution 注意
小さな勝率変化を気にしすぎないこと。アマチュア愛好家にとって、1-2%の勝率差は通常実際的な意味はありません。
:::

## まとめ

AI時代がもたらした最も重要な変化は、具体的な打ち方ではなく、心構えの転換かもしれません：

- **「権威を受け入れる」から「独立して考える」へ**
- **「定石を暗記する」から「原理を理解する」へ**
- **「正解を追い求める」から「多様な可能性を受け入れる」へ**

この心構えの転換は、囲碁を学ぶ上でも、人生に向き合う上でも、ポジティブな示唆を与えてくれます。
