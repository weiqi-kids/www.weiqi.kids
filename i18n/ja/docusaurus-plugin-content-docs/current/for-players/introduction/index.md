# 入門編

囲碁の世界へようこそ！このセクションでは、ゼロから始めて、しっかりとした囲碁の基礎を築いていきます。

## このセクションの内容

### 囲碁のルール

囲碁のルールはシンプルですが奥深いものです。学ぶ内容：

- **気**：石の生命線
- **取り**：相手の石を取る方法
- **禁着点**：打てない場所
- **コウ**：特殊な循環ルール
- **眼と活き**：石を生かす方法
- **勝敗判定**：地計算法と数子法

[囲碁のルールを学ぶ →](./rules)

### 基本用語

囲碁の専門用語を理解することで：

- 棋譜解説を読めるようになる
- 他の囲碁愛好家と交流できる
- 教材の内容を理解できる

[囲碁用語を知る →](./terminology)

### 対局マナー

囲碁は勝負だけでなく、文化でもあります。学ぶ内容：

- 着手の礼儀
- 対局中の態度
- 勝負の品格

[対局マナーを知る →](./etiquette)

### 序盤の考え方

実際に打ち始める前に、正しい考え方を身につけましょう：

- 布石の大きな方向性
- 隅、辺、中央の価値
- 一般的な序盤の着点の特徴

[序盤の考え方を学ぶ →](./first-10-moves)

## 学習アドバイス

:::tip 初心者の方へ
勝つことを焦らないでください。囲碁を学び始めたばかりの時に最も大切なのは：

1. **ルールを確実に理解する** - 特に「気」と「眼」の概念
2. **たくさん打つ** - 9路盤の小さな碁盤で練習し、経験を積む
3. **過程を楽しむ** - 囲碁は一生の趣味、ゆっくりと
:::

## 学習順序

以下の順序で学習することをお勧めします：

| 順序 | テーマ | ポイント |
|:---:|------|------|
| 1 | ルール | まずルールをしっかり理解する |
| 2 | 用語 | 基本用語を知る |
| 3 | マナー | 正しい態度を身につける |
| 4 | 序盤 | 布石の考え方を確立する |

準備はできましたか？[囲碁のルール](./rules)から始めましょう！
