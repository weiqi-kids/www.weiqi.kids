# 囲碁AIの発展まとめ

## なぜAIが囲碁を変えたのか？

2016年3月、Google DeepMindのAlphaGoが世界チャンピオンのイ・セドルを4:1で破り、この試合は囲碁界だけでなく世界全体に衝撃を与えました。囲碁、この人類の知性の最後の砦と考えられていたゲームが、ついに人工知能に攻略されたのです。

しかしこれは終わりではなく、新時代の始まりでした。

### AIがもたらした変化

**1. 「最善手」の再定義**

AI出現以前、人類は数千年の囲碁の知恵を蓄積し、「正しい」とされる打ち方の体系を形成していました。しかしAIの出現は多くの伝統的観念を覆しました：

- 序盤はもはや星や小目にこだわらず、三々点が主流に
- 早期の肩衝きなどの「禁手」が再評価された
- 全盤効率の思考方法が局部定石に取って代わった

**2. 学習方法の革命**

過去、棋士がトップレベルに達するには数十年の時間が必要でした。今では：

- 若い棋士は24時間AIと対局できる
- 各手に客観的な数値評価が得られる
- 検討時に最善手と自分の手の差を見られる

**3. 棋芸の天井の向上**

AIは人類に囲碁のより高い境地を見せてくれました：

- AlphaGoの勝率評価は人間の直感をはるかに超える
- 現代のプロ棋士の棋力は全体的に向上
- 対局の精彩さと深さが向上

### 本章の内容

本章では囲碁の古代から現代までの発展の歴史と、AIがいかに短期間でこの歴史ある遊戯を変えたかを振り返ります。

#### [人類の囲碁発展史](/docs/evolution/human-history)

中国古代の起源伝説から、日本四大棋家の黄金時代、韓国と中国の台頭まで、人類がいかに数千年にわたってこの知的ゲームを発展させてきたかを理解します。

#### [AI囲碁発展史](/docs/evolution/ai-history)

2015年にAlphaGoが初めてプロ棋士に勝ってから、今日のKataGoなどのオープンソースAIの普及まで、人工知能がいかに短期間で人類数千年の知恵の蓄積を超えたかを目撃します。
