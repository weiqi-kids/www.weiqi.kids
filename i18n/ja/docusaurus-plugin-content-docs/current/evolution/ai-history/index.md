# AI囲碁発展史

長い間、囲碁は人工知能が最も攻略困難なゲームと考えられていました。碁盤上の19×19 = 361の交点、各点に着手可能で、変化の数は宇宙の原子の総数を超えます（約10^170種類の可能な棋局）。従来の総当たり探索法は囲碁の前では完全に無力でした。

しかし2015年から2017年にかけて、DeepMindのAlphaGoシリーズがすべてを変えました。この革命は囲碁に影響を与えただけでなく、AI分野全体の発展を推進しました。

## なぜ囲碁はこれほど難しいのか？

### 膨大な探索空間

チェスを例にとると、平均して各手に約35通りの合法手があり、1局は約80手です。一方、囲碁は平均して各手に約250通りの合法手があり、1局は約150手です。これは囲碁の探索空間がチェスより数百桁大きいことを意味します。

### 評価困難な局面

チェスの各駒には明確な価値があり（クイーン9点、ルーク5点など）、簡単な公式で局面を評価できます。しかし囲碁では、1つの石の価値は周囲の石との関係で決まり、簡単な評価方法がありません。

一塊の石は生きているか死んでいるか？一片の勢力は何目の価値があるか？これらの問題は人間の専門家にとっても、しばしば深い計算と判断が必要です。

### 初期囲碁プログラムの困難

AlphaGo以前、最強の囲碁プログラムはアマチュア5-6段程度で、プロ棋手とは大きな差がありました。これらのプログラムは主に「モンテカルロ木探索」（MCTS）法を使用し、大量のランダムシミュレーションで局面を評価していました。

しかしこの方法には明らかな限界がありました：ランダムシミュレーションは囲碁の戦略的思考を捉えられず、プログラムはしばしば人間から見ると非常に愚かなミスを犯しました。

## AI囲碁の2つの時代

### [AlphaGo時代（2015-2017）](/docs/evolution/ai-history/alphago-era)

この時代はAlphaGoが樊麾を破ることから始まり、AlphaZero論文の発表で終わります。DeepMindはわずか2年で、プロ棋手に勝つことから人類の限界を超えるまでの飛躍を実現しました。

主なマイルストーン：
- 2015.10：樊麾に勝利（初のプロ棋手への勝利）
- 2016.03：イ・セドルに勝利（4:1）
- 2017.01：Masterがオンラインで60連勝
- 2017.05：柯潔に勝利（3:0）
- 2017.10：AlphaZero発表

### [KataGo時代（2019-現在）](/docs/evolution/ai-history/katago-era)

AlphaGo引退後、オープンソースコミュニティが松明を引き継ぎました。KataGo、Leela Zeroなどのオープンソースのおかげで、誰もがトップレベルの囲碁エンジンを使えるようになり、囲碁の学習と訓練方法が根本的に変わりました。

この時代の特徴：
- AIツールの民主化
- プロ棋手のAI訓練の普及
- 人間の棋風のAI化
- 囲碁全体のレベル向上

## AIがもたらした認識の衝撃

### 「正しい打ち方」の再定義

AI出現以前、人類は数千年かけて、「正しい」とされる囲碁理論体系を構築しました。しかしAIの多くの打ち方は人間の伝統的認識に反していました：

- **三々点**：伝統的観念では序盤での直接三々は「俗手」、AIはしばしばこう打つ
- **肩衝き**：過去「悪手」とされた肩衝きが、ある局面ではAIにより最善と証明された
- **接近戦**：AIは近接戦闘を好み、人間の伝統的な「攻撃は遠くから」の理念と異なる

### 人間の限界と潜在能力

AIの出現は人類に自身の限界を認識させましたが、同時に人類の潜在能力も示しました。

AIの助けにより、若い棋手の成長速度は大幅に加速しました。過去10年かかったレベルに、今では3-5年で到達できる可能性があります。囲碁全体のレベルが向上しています。

### 囲碁の未来

AIが囲碁を無意味にするのではないかと心配する人もいました——永遠にAIに勝てないなら、なぜ打つのか？

しかし事実はこの心配が杞憂だったことを証明しました。AIは囲碁を終わらせたのではなく、囲碁の新紀元を開いたのです。人間同士の対局は依然として創造性、感情、予測不可能性に満ちています——これらこそが囲碁を面白くする本質です。
