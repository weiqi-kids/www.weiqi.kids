# 샤다펑

**LINE@ 개발 엔지니어**

## 소개

샤다펑은 LINE@ 개발 전문가로, 호치베이비협회의 LINE 공식 계정의 인터랙티브 기능을 개발했습니다. 미션 클리어, 태스크 리스트 등 게이미피케이션 학습 시스템을 구현했습니다.

## 협업 작품

### LINE@ 호치베이비

![LINE@ 인터페이스](/img/partners/da-feng-line.jpg)

호치베이비의 LINE 공식 계정을 개발하여 다음 기능을 제공합니다:

- **오늘의 미션**: 매일 바둑 학습 과제
- **태스크 리스트**: 학습 진도 추적
- **미션 클리어**: 지속적인 학습 장려
- **최신 소식**: 협회 행사 푸시 알림

## 기술적 특징

- LINE Messaging API 연동
- 게이미피케이션 학습 메커니즘
- 인터랙티브 메뉴 디자인
- 자동 메시지 푸시
