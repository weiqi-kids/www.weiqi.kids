# 창립 회원

대만호치베이비협회는 바둑을 사랑하는 다양한 분야의 전문가들이 함께 설립했습니다. 바둑 문화에 대한 열정과 각자의 전문성을 바탕으로 대만에서의 바둑 발전에 힘쓰고 있습니다.

## 이사장

- [장라오후이](./lightman-chang) - 대만호치베이비협회
