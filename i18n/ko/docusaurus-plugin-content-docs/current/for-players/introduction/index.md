# 입문편

바둑의 세계에 오신 것을 환영합니다! 이 섹션에서는 처음부터 차근차근 탄탄한 바둑 기초를 쌓을 수 있도록 안내합니다.

## 이 섹션의 내용

### 바둑 규칙

바둑의 규칙은 간단하지만 심오합니다. 다음을 배우게 됩니다:

- **활로**: 돌의 생명선
- **따냄**: 상대의 돌을 잡는 방법
- **착수금지점**: 둘 수 없는 곳
- **패**: 특수한 반복 규칙
- **눈과 삶**: 돌을 살리는 방법
- **승부 판정**: 집계산법과 돌계산법

[바둑 규칙 배우기 →](./rules)

### 기본 용어

바둑 전문 용어를 익히면:

- 기보 해설을 이해할 수 있습니다
- 다른 기사들과 소통할 수 있습니다
- 교재 내용을 이해할 수 있습니다

[바둑 용어 익히기 →](./terminology)

### 대국 예절

바둑은 승부만이 아니라 문화이기도 합니다. 다음을 배웁니다:

- 착수의 예절
- 대국의 태도
- 승부의 품격

[대국 예절 알아보기 →](./etiquette)

### 포석 개념

본격적으로 바둑을 두기 전에 올바른 개념을 세웁니다:

- 포석의 큰 방향
- 귀, 변, 중앙의 가치
- 흔한 포석 위치의 특성

[포석 개념 배우기 →](./first-10-moves)

## 학습 조언

:::tip 초보자를 위한 말씀
이기려고 조급해하지 마세요. 바둑을 처음 배울 때 가장 중요한 것은:

1. **규칙을 확실히 이해하기** - 특히 '활로'와 '눈'의 개념
2. **많이 두기** - 9로 소형 바둑판에서 연습하며 경험 쌓기
3. **과정을 즐기기** - 바둑은 평생의 취미입니다, 천천히 하세요
:::

다음 순서로 학습하시는 것을 권장합니다:
