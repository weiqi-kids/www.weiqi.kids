# 바둑 기사를 위한 자료

바둑 기사 전용 섹션에 오신 것을 환영합니다! 바둑을 처음 접하는 초보자든, 기력 향상을 원하는 애호가든, 여기에서 적합한 학습 자료를 찾으실 수 있습니다.

## 이 섹션의 대상

- **완전 초보자**: 바둑을 처음 접하며 기초부터 배우고 싶은 분
- **초급자**: 기본 규칙은 알고 있으나 올바른 개념을 세우고 싶은 분
- **아마추어 기사**: 어느 정도 기초가 있고 체계적으로 기력을 높이고 싶은 분
- **복귀 기사**: 예전에 바둑을 배웠다가 다시 시작하고 싶은 분

## 내용 개요

### 입문편

가장 기초적인 규칙부터 시작하여 바둑의 세계를 안내합니다:

- [바둑 규칙](/docs/learn/introduction/rules) - 활로, 따냄, 착수금지점, 패, 눈과 삶
- [기본 용어](/docs/learn/introduction/terminology) - 바둑판, 착수, 모양, 대국 관련 용어
- [대국 예절](/docs/learn/introduction/etiquette) - 바둑의 예절과 태도
- [포석 첫 10수](/docs/learn/introduction/first-10-moves) - 포석의 기본 방향과 개념

### AI 시대

인공지능이 바둑을 어떻게 변화시켰는지 알아봅니다:

- [AI가 가져온 변화](/docs/learn/ai-era/changes) - 기풍과 관념의 전환
- [AI로 바둑 배우기](/docs/learn/ai-era/learning-with-ai) - 도구를 활용한 실력 향상

## 학습 경로 제안

### 1단계: 기초 다지기 (1-3개월)

1. 규칙을 숙지하고, 특히 '활로'와 '눈'의 개념을 확실히 이해합니다
2. 기본 용어를 익혀 기보 해설을 이해할 수 있도록 합니다
3. 9로 또는 13로 소형 바둑판에서 연습합니다
4. 올바른 대국 예절을 기릅니다

### 2단계: 기감 형성 (3-6개월)

1. 포석의 큰 방향을 이해합니다
2. 간단한 사활 문제를 풀어봅니다
3. 19로 대형 바둑판에서 대국을 시작합니다
4. 매 대국 후 간단히 복기합니다

### 3단계: 체계적 향상 (6개월 이상)

1. 기본 정석과 흔한 변화를 학습합니다
2. 형세 판단 능력을 기릅니다
3. AI 도구를 활용하여 자신의 대국을 분석합니다
4. 프로 기사의 대국을 감상하며 대국관을 기릅니다

:::tip 학습 조언
바둑의 실력 향상에는 시간과 인내가 필요합니다. 많은 지식을 급하게 배우려 하기보다 먼저 기초를 탄탄히 다지세요. 매일 한 판씩 두고, 몇 문제씩 사활을 풀며, 꾸준히 하는 것이 무엇보다 중요합니다.
:::
