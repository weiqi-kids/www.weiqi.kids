# 바둑 용어

바둑 용어를 배우면 기보를 이해하고, 해설을 파악하며, 다른 기사들과 소통하는 데 도움이 됩니다.

## 바둑판 용어

### 바둑판의 중요 위치

| 용어 | 위치 | 설명 |
|------|------|------|
| **천원** | 바둑판 정중앙 | 19로 바둑판의 10선 교점(10,10) |
| **화점** | 네 귀 근처의 검은 점 | 변에서 각 4줄 떨어진 위치(4,4) |
| **소목** | 귀의 흔한 착점 | (3,4) 또는 (4,3) 위치 |
| **삼삼** | 귀의 가장 깊은 곳 | (3,3) 위치, 직접 귀를 취하는 수법 |
| **목외** | 귀의 다소 높은 위치 | (3,5) 또는 (5,3) 위치 |
| **고목** | 귀의 높은 위치 | (4,5) 또는 (5,4) 위치 |

### 바둑판의 구역

| 용어 | 설명 |
|------|------|
| **귀** | 바둑판 네 귀 구역, 보통 4선 이내를 말함 |
| **변** | 바둑판 네 변 구역, 귀 사이의 부분 |
| **중앙** | 바둑판 중앙 구역, '중복' 또는 '중원'이라고도 함 |

### 위치의 높낮이

| 용어 | 설명 |
|------|------|
| **1선** | 가장 바깥 변선 (사선) |
| **2선** | 두 번째 선 (패선) |
| **3선** | 세 번째 선 (실리선) |
| **4선** | 네 번째 선 (세력선) |

:::note 격언
"1선에서 살려면 나무에서 물고기 잡기, 2선으로 머리 내밀면 이익 없다" — 1, 2선은 불리한 위치입니다.
:::

## 착수 용어

### 기본 동작

| 용어 | 설명 |
|------|------|
| **착수** | 돌을 바둑판에 놓는 것 |
| **따냄** | 활로가 없는 상대 돌을 들어내는 것 |
| **버림** | 의도적으로 일부 돌을 구하지 않는 것 |

### 공격 관련

| 용어 | 설명 |
|------|------|
| **단수 / 아타리** | 상대 돌의 활로를 하나만 남기는 것 |
| **양단수** | 두 덩어리를 동시에 단수 쳐서 상대가 한 쪽만 구할 수 있게 하는 것 |
| **축** | 연속 단수로 상대가 도망갈 수 없는 잡는 방법 |
| **장문** | 느슨하게 막아 도망가면 잡히게 하는 것 |

### 돌의 이동 방식

| 용어 | 설명 | 모양 |
|------|------|------|
| **늘임** | 직선으로 연결하여 나가는 것 | 직선 연장 |
| **건너뜀** | 한 칸 건너뛰는 것(같은 선상) | 중간에 한 칸 빔 |
| **날일자** | 대각선 방향으로 이동, 중간에 한 칸 빔 | 장기의 상과 비슷 |
| **큰날일자** | 대각선 방향으로 더 멀리 이동 | 날일자보다 더 멀리 |
| **마늘모** | 대각선으로 붙어 있음(사선으로 인접) | 비스듬히 연결 |
| **한칸뜀** | 같은 선상에서 한 칸 건너뛰기(건너뜀과 동일) | 가로 또는 세로 |

### 접촉전 용어

| 용어 | 설명 |
|------|------|
| **젖힘** | 상대 돌의 사선에서 막는 것 |
| **끊음** | 상대 돌의 연결을 자르는 것 |
| **이음** | 자기 돌을 연결하는 것 |
| **막음** | 상대의 진행을 막는 것 |
| **협공** | 양쪽에서 상대 돌을 끼우는 것 |
| **붙임** | 상대 돌에 바짝 붙여 두는 것 |
| **밀착** | 바짝 붙여 두는 것, 보통 상대 돌 옆에 |
| **누름** | 위에서 눌러 상대를 낮게 만드는 것 |
| **받침** | 상대 돌 아래에서 받치는 것 |
| **찍음** | 핵심 위치에 한 수 두는 것(흔히 눈자리나 급소에) |
| **들여다봄** | 상대의 약점을 향해 한 수로 위협하는 것 |

## 모양 용어

### 좋은 모양

| 용어 | 설명 |
|------|------|
| **호구** | 두 돌이 대각선으로, 중간에 한 점 비어 호구 형성 |
| **쌍** | 두 돌이 연결된 기본 모양 |
| **두터움** | 약점이 없는 매우 안정된 모양 |
| **가벼움** | 탄력 있는 모양, 공격받아도 괜찮음 |

### 기본 개념

| 용어 | 설명 |
|------|------|
| **눈** | 자기 돌로 둘러싸인 빈 점 |
| **활로** | 돌이나 돌덩어리 주위의 빈 점(생명선) |
| **단점** | 돌 연결의 약점, 끊어질 수 있는 곳 |
| **연결** | 돌이 연결되어 하나가 됨 |

### 나쁜 모양

| 용어 | 설명 |
|------|------|
| **우형** | 효율이 나쁜 모양 |
| **뭉침** | 돌이 뭉쳐 있어 효율이 낮음 |
| **빈삼각** | 세 돌이 빈삼각을 이룸, 전형적인 우형 |
| **찢김** | 모양이 나뉘어 공격받기 쉬움 |

## 대국 용어

### 수의 순서

| 용어 | 설명 |
|------|------|
| **선수** | 이 수를 두고 나서 내가 계속 주도권을 갖는 것 |
| **후수** | 이 수를 두고 나서 상대에게 먼저 둘 차례를 넘기는 것 |
| **선수 잡음** | 국부를 다 마무리하지 않고 다른 큰 곳으로 전환 |
| **선수 이득** | 선수로 얻는 이익 |

### 대국 단계

| 용어 | 설명 |
|------|------|
| **포석** | 초반 단계, 양측이 큰 곳을 차지함 |
| **중반** | 전투가 가장 치열한 단계 |
| **끝내기** | 종반 단계, 경계를 정리함 |
| **수습** | 끝내기를 진행, 경계를 마무리함 |

### 국면 판단

| 용어 | 설명 |
|------|------|
| **우세** | 국면이 앞서 있음 |
| **열세** | 국면이 뒤처져 있음 |
| **세바둑** | 양측 차이가 매우 작은 국면 |
| **대국관** | 전체 국면에 영향을 미치는 수법이나 판단 |

### 대국 결과

| 용어 | 설명 |
|------|------|
| **흑 X집 승** | 흑이 X집 차이로 승리 |
| **백 X집 승** | 백이 X집 차이로 승리 |
| **비김** | 양측 집수가 같음(매우 드묾) |
| **불계승** | 집을 세지 않는 승리(상대 기권 또는 포기) |

## 고급 용어

이 용어들은 고급 학습에서 자주 만나게 됩니다:

| 용어 | 설명 |
|------|------|
| **정석** | 초반에 양측 최선의 응대가 정해진 수법 |
| **수근** | 교묘한 수법, 멋진 한 수 |
| **속수** | 평범하고 효율이 낮은 수법 |
| **느린수** | 급하지 않아 상대에게 이익을 주는 수법 |
| **급소** | 반드시 바로 두어야 하는 중요한 위치 |
| **큰 곳** | 가치가 매우 큰 위치 |
| **두터운 세력** | 두터운 세력으로 외부에 영향력이 있음 |
| **실리** | 확정된 영역 |
| **모양** | 아직 완성되지 않았지만 잠재력 있는 영역 |

:::tip 학습 조언
모든 용어를 한 번에 외울 필요는 없습니다. 먼저 '착수 용어'의 기본 용어에 익숙해지고, 나머지는 실전에서 천천히 배우시면 됩니다.
:::
