# How to Use the Weiqi.Kids Website

Welcome to our website! Here you'll find resources organized for Go players and AI engineers. You can access different topics from the left sidebar.

## Section Overview

- **For Go Players**: Organized by skill level into Introduction, Pre-Dan, and Dan-Level sections.
- **For Engineers**: Provides KataGo architecture, research background, and source code information.
- **Evolution**: Covers the history of human Go development and AI Go progress.
