# Terminologia do Go

Aprender a terminologia do Go ajuda voce a entender comentarios de partidas, compreender explicacoes e comunicar-se com outros jogadores.

## Terminologia do Tabuleiro

### Posicoes Importantes do Tabuleiro

| Termo | Posicao | Descricao |
|------|------|------|
| **Tengen** | Centro exato do tabuleiro | Intersecao da 10a linha no tabuleiro 19x19 (10,10) |
| **Hoshi (Estrela)** | Pontos marcados perto dos cantos | Posicoes a 4 linhas de cada borda (4,4) |
| **Komoku** | Ponto de abertura comum no canto | Posicao (3,4) ou (4,3) |
| **San-san (3-3)** | Ponto mais profundo do canto | Posicao (3,3), jogada direta no canto |
| **Mokuhazushi** | Posicao alta no canto | Posicao (3,5) ou (5,3) |
| **Takamoku** | Posicao mais alta no canto | Posicao (4,5) ou (5,4) |

### Regioes do Tabuleiro

| Termo | Descricao |
|------|------|
| **Canto** | Regioes dos quatro cantos do tabuleiro, geralmente dentro da 4a linha |
| **Lateral** | Regioes das bordas entre os cantos |
| **Centro** | Regiao central do tabuleiro, tambem chamada "barriga" |

### Altura das Posicoes

| Termo | Descricao |
|------|------|
| **Primeira linha** | Linha mais externa (linha da morte) |
| **Segunda linha** | Segunda linha (linha da derrota) |
| **Terceira linha** | Terceira linha (linha do territorio) |
| **Quarta linha** | Quarta linha (linha da influencia) |

:::note Proverbio
"Primeira linha busca vida como peixe na arvore, segunda linha sai com medo de nao ter suficiente" -- Primeira e segunda linhas sao posicoes desfavoraveis.
:::

## Terminologia de Jogadas

### Acoes Basicas

| Termo | Descricao |
|------|------|
| **Colocar pedra** | Colocar uma pedra no tabuleiro |
| **Capturar** | Remover pedras do oponente sem liberdades |
| **Sacrificar** | Abandonar intencionalmente certas pedras |

### Relacionados a Ataque

| Termo | Descricao |
|------|------|
| **Atari (Xeque)** | Deixar pedras do oponente com apenas uma liberdade |
| **Atari duplo** | Dar atari em dois grupos simultaneamente, oponente so pode salvar um |
| **Escada (Shicho)** | Ataris consecutivos onde oponente nao pode escapar |
| **Rede (Geta)** | Cercado frouxo onde mesmo correndo sera capturado |

### Formas de Movimento das Pedras

| Termo | Descricao | Forma |
|------|------|------|
| **Nobi** | Conectar em linha reta | Extensao linear |
| **Tobi (Pulo)** | Pular uma intersecao (na mesma linha) | Um espaco entre |
| **Keima (Salto de cavalo)** | Movimento diagonal, um espaco entre | Similar ao cavalo no xadrez |
| **Ogeima (Salto grande)** | Movimento diagonal, distancia maior | Maior que keima |
| **Kosumi** | Diagonal adjacente | Conexao diagonal |
| **Ikken-tobi** | Pulo de um espaco na mesma linha | Horizontal ou vertical |

### Terminologia de Contato

| Termo | Descricao |
|------|------|
| **Hane** | Virar no canto da pedra do oponente |
| **Kireru (Cortar)** | Separar pedras conectadas do oponente |
| **Tsugu (Conectar)** | Conectar suas proprias pedras |
| **Osae (Bloquear)** | Bloquear o caminho do oponente |
| **Hasami (Pincar)** | Atacar dos dois lados |
| **Tsuke (Encostar)** | Jogar adjacente a pedra do oponente |
| **Nobi-tsuke** | Jogar colado, geralmente ao lado da pedra do oponente |
| **Osu (Empurrar)** | Empurrar de cima, forcando oponente para baixo |
| **Uchi-komi** | Jogar por baixo da pedra do oponente |
| **Tsume (Ponto vital)** | Jogar em posicao chave (frequentemente no olho ou ponto vital) |
| **Nozoki (Ameacar corte)** | Jogar ameacando um ponto fraco do oponente |

## Terminologia de Formas

### Boas Formas

| Termo | Descricao |
|------|------|
| **Tobi-gashira (Cabeca de tigre)** | Duas pedras diagonais com espaco no meio, formando "boca de tigre" |
| **Narabi** | Duas pedras conectadas, forma basica |
| **Atsui (Espesso)** | Forma sem fraquezas, muito solida |
| **Karui (Leve)** | Forma flexivel, nao tem medo de ataque |

### Conceitos Basicos

| Termo | Descricao |
|------|------|
| **Me (Olho)** | Ponto vazio cercado por pedras proprias |
| **Iki (Liberdade)** | Pontos vazios ao redor das pedras (linha de vida) |
| **Kiri-me (Ponto de corte)** | Fraqueza na conexao, lugar que pode ser cortado |
| **Tsunagaru (Conexao)** | Pedras conectadas formando uma unidade |

### Formas Ruins

| Termo | Descricao |
|------|------|
| **Guzumi (Forma estupida)** | Forma ineficiente |
| **Katachi ga warui** | Pedras amontoadas, eficiencia baixa |
| **Aki-sankaku (Triangulo vazio)** | Tres pedras formando triangulo vazio, forma estupida tipica |
| **Sabaki ga warui** | Forma dividida, facil de atacar |

## Terminologia de Partida

### Ordem das Jogadas

| Termo | Descricao |
|------|------|
| **Sente** | Apos esta jogada, voce continua tendo a iniciativa |
| **Gote** | Apos esta jogada, precisa ceder a iniciativa ao oponente |
| **Tenuki** | Nao terminar localmente, mudar para outro lugar grande |
| **Sente no riten** | Vantagem obtida em sente |

### Fases da Partida

| Termo | Descricao |
|------|------|
| **Fuseki (Abertura)** | Fase inicial, ambos ocupam pontos importantes |
| **Chuban (Meio-jogo)** | Fase mais intensa de luta |
| **Yose (Finalizacao)** | Fase final, definindo fronteiras |
| **Shusoku** | Fazer o yose, fechar as fronteiras |

### Avaliacao da Posicao

| Termo | Descricao |
|------|------|
| **Yusei (Vantagem)** | Posicao a frente |
| **Rettai (Desvantagem)** | Posicao atras |
| **Hosoki (Jogo apertado)** | Posicao com pequena diferenca |
| **Taikyoku** | Jogada ou avaliacao que afeta toda a posicao |

### Resultados da Partida

| Termo | Descricao |
|------|------|
| **Preto vence por X pontos** | Preto ganha por X pontos |
| **Branco vence por X pontos** | Branco ganha por X pontos |
| **Jigo (Empate)** | Ambos com mesma pontuacao (muito raro) |
| **Chuban-gachi (Vitoria no meio-jogo)** | Vitoria sem contar pontos (oponente resignou) |

## Terminologia Avancada

Estes termos serao encontrados frequentemente no aprendizado avancado:

| Termo | Descricao |
|------|------|
| **Joseki** | Sequencias de jogadas otimas fixas na abertura |
| **Tesuji** | Jogada habilidosa, movimento brilhante |
| **Zokusuji** | Jogada comum, movimento ineficiente |
| **Nurui-te** | Jogada lenta, que da vantagem ao oponente |
| **Kyusho** | Posicao que deve ser jogada imediatamente |
| **Oba** | Posicao de grande valor |
| **Atsumi** | Influencia solida com poder externo |
| **Jisseki** | Territorio confirmado |
| **Moyo** | Territorio potencial ainda nao completo |

:::tip Sugestao de Aprendizado
Nao precisa memorizar todos os termos de uma vez. Sugerimos primeiro se familiarizar com os termos basicos em "Terminologia de Jogadas", e aprender os outros gradualmente na pratica.
:::
