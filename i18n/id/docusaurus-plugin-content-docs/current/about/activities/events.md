# Acara Go

Dokumentasi acara Go yang diselenggarakan dan diikuti oleh asosiasi.

## Karnaval Go Keluarga Good Go Baby Cup ke-1

**Tanggal**: Juni 2024
**Lokasi**: SMA Heping

![Lokasi Karnaval](/img/activities/carnival/venue.jpg)

Karnaval Go Keluarga Good Go Baby Cup ke-1 menggabungkan turnamen Go dengan kegiatan keluarga, memungkinkan seluruh keluarga menikmati kesenangan Go bersama.

### Sorotan Acara

- Turnamen Go grup
- Permainan interaktif orangtua-anak
- Stan pengalaman Go
- Upacara penghargaan
