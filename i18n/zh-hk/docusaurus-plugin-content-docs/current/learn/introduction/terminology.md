# 圍棋術語

學習圍棋術語，可以幫你睇得明棋譜、理解解說，同埋同其他棋友交流。

## 棋盤術語

### 棋盤嘅重要位置

| 術語 | 位置 | 說明 |
|------|------|------|
| **天元** | 棋盤正中央 | 19 路棋盤嘅第 10 線交點（10,10） |
| **星位** | 四個角落附近嘅黑點 | 距離邊線各 4 條線嘅位置（4,4） |
| **小目** | 角落嘅常見落子點 | (3,4) 或 (4,3) 嘅位置 |
| **三三** | 角落最深處 | (3,3) 嘅位置，直接攞角嘅下法 |
| **目外** | 角落較高嘅位置 | (3,5) 或 (5,3) 嘅位置 |
| **高目** | 角落嘅高位 | (4,5) 或 (5,4) 嘅位置 |

### 棋盤嘅區域

| 術語 | 說明 |
|------|------|
| **角** | 棋盤四個角落區域，通常指 4 線以內 |
| **邊** | 棋盤四邊區域，角落之間嘅部分 |
| **中腹** | 棋盤中央區域，亦叫「中央」或「腹地」 |

### 位置嘅高低

| 術語 | 說明 |
|------|------|
| **一線** | 最外面嘅邊線（死亡線） |
| **二線** | 第二條線（敗線） |
| **三線** | 第三條線（實地線） |
| **四線** | 第四條線（勢力線） |

:::note 俗諺
「一線求活緣木魚，二線出頭恐不盈」—— 一、二線係唔利嘅位置。
:::

## 落子術語

### 基本動作

| 術語 | 說明 |
|------|------|
| **落子** | 將棋子放到棋盤上 |
| **提子** | 將冇氣嘅對方棋子攞走 |
| **棄子** | 故意放棄某啲棋子唔救 |

### 攻擊相關

| 術語 | 說明 |
|------|------|
| **打吃 / 叫吃** | 令對方棋子淨返一口氣 |
| **雙叫吃** | 同時叫吃兩塊棋，對方淨係救得一塊 |
| **征子（征吃）** | 連續叫吃，對方走唔甩嘅食法 |
| **枷** | 鬆鬆咁封住對方，走都會被食 |

### 棋子嘅移動方式

| 術語 | 說明 | 形狀 |
|------|------|------|
| **長** | 沿住直線連接出去 | 直線延伸 |
| **跳** | 隔一格跳出（同線上） | 中間空一格 |
| **飛** | 對角方向移動，中間空一格 | 類似象棋嘅象 |
| **大飛** | 對角方向移動，距離更遠 | 比飛更遠 |
| **尖** | 對角緊靠（斜向相鄰） | 斜住連接 |
| **關** | 同線上隔一格跳（同跳） | 橫向或縱向 |

### 接觸戰術語

| 術語 | 說明 |
|------|------|
| **扳** | 喺對方棋子斜角頂住 |
| **斷** | 切斷對方棋子嘅連接 |
| **接** | 將自己嘅棋子連接埋 |
| **擋** | 擋住對方嘅去路 |
| **夾** | 由兩邊夾住對方嘅棋子 |
| **靠** | 緊貼住對方嘅棋子落 |
| **貼** | 緊挨住落，通常喺對方棋子隔籬 |
| **壓** | 由上方壓住對方，令對方變低 |
| **托** | 喺對方棋子下方托住 |
| **點** | 喺關鍵位置落一子（常喺眼位或要點） |
| **刺** | 向住對方嘅弱點落一子威脅 |

## 棋形術語

### 好形

| 術語 | 說明 |
|------|------|
| **虎** | 兩子對角，中間空一點，形成虎口 |
| **雙** | 兩子連接之後嘅基本形狀 |
| **厚實** | 棋形冇弱點，非常穩固 |
| **輕靈** | 棋形有彈性，唔驚被攻 |

### 基本概念

| 術語 | 說明 |
|------|------|
| **眼** | 被己方棋子圍住嘅空點 |
| **氣** | 棋子或棋塊周圍嘅空點（生命線） |
| **斷點** | 棋子連接嘅弱點，可能被切斷嘅地方 |
| **連接** | 棋子相連形成一體 |

### 唔好嘅形

| 術語 | 說明 |
|------|------|
| **愚形** | 效率差嘅棋形 |
| **凝形** | 棋子迫埋一齊，效率唔高 |
| **空三角** | 三子圍成空三角，典型嘅愚形 |
| **裂形** | 棋形被分開，容易被攻擊 |

## 對局術語

### 手嘅順序

| 術語 | 說明 |
|------|------|
| **先手** | 落完呢手之後，輪到我方繼續主導 |
| **後手** | 落完呢手之後，需要等對方先動 |
| **搶先** | 喺局部唔收完，轉去其他大場 |
| **先手利** | 先手得到嘅著數 |

### 對局階段

| 術語 | 說明 |
|------|------|
| **佈局** | 開局階段，雙方佔領大場 |
| **中盤** | 戰鬥最激烈嘅階段 |
| **官子** | 終局階段，收拾邊界 |
| **收官** | 進行官子，將邊界收好 |

### 局面判斷

| 術語 | 說明 |
|------|------|
| **優勢** | 局面領先 |
| **劣勢** | 局面落後 |
| **細棋** | 雙方差距好細嘅局面 |
| **大局** | 影響整體局勢嘅下法或判斷 |

### 對局結果

| 術語 | 說明 |
|------|------|
| **黑勝 X 目** | 黑方贏 X 目 |
| **白勝 X 目** | 白方贏 X 目 |
| **和棋** | 雙方目數相同（非常罕見） |
| **中盤勝** | 唔計算目數嘅勝利（對方認輸或棄權） |

## 進階術語

呢啲術語喺進階學習嗰陣會經常遇到：

| 術語 | 說明 |
|------|------|
| **定式** | 開局時雙方最佳應對嘅固定下法 |
| **手筋** | 巧妙嘅手法，精彩嘅一手 |
| **俗手** | 普通、效率唔高嘅下法 |
| **緩著** | 唔夠緊迫、令對方得利嘅下法 |
| **急所** | 必須即刻落嘅重要位置 |
| **大場** | 價值好大嘅位置 |
| **厚勢** | 厚實嘅勢力，對外有影響力 |
| **實地** | 確定圍住嘅地盤 |
| **模樣** | 未完成但有潛力嘅地盤 |

唔使一次記晒所有術語。建議先熟悉「落子術語」入面嘅基本用詞，其他嘅喺實戰入面慢慢學就得。
:::
