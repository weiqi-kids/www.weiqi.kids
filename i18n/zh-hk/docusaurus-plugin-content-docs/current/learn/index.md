# 畀圍棋棋友

歡迎嚟到圍棋棋友專區！無論你係啱啱接觸圍棋嘅新手，定係想精進棋藝嘅愛好者，呢度都有適合你嘅學習資源。

## 呢個區塊適合邊個

- **完全新手**：從未接觸過圍棋，想由零開始學習
- **初學者**：已經知道基本規則，想建立正確嘅觀念
- **業餘棋友**：有一定基礎，希望系統性咁提升棋力
- **回歸棋友**：曾經學過圍棋，想重新拾起呢項興趣

## 學習路徑建議

### 階段一：打好基礎（1-3 個月）

1. 熟讀規則，確保理解「氣」同「眼」嘅概念
2. 學習基本術語，能夠睇得明棋譜解說
3. 喺 9 路或 13 路細棋盤上練習
4. 養成良好嘅對弈禮儀

### 階段二：建立棋感（3-6 個月）

1. 理解開局嘅大方向
2. 練習簡單嘅死活題
3. 開始喺 19 路大棋盤落子
4. 每盤棋之後簡單復盤

### 階段三：系統提升（6 個月以上）

1. 學習基本定式同常見變化
2. 培養形勢判斷能力
3. 善用 AI 工具分析自己嘅棋局
4. 欣賞職業棋士嘅對局，培養大局觀

:::tip 學習建議
圍棋嘅進步需要時間同耐心。比起急住學好多知識，不如先將基礎打穩。每日落一盤棋、做幾道死活題，持之以恆比乜嘢都重要。
:::
