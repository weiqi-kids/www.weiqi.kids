# 入門篇

歡迎踏入圍棋嘅世界！呢個區塊會帶你由零開始，建立扎實嘅圍棋基礎。

## 本區塊內容

### 圍棋規則

圍棋嘅規則簡單但深奧。你會學到：

- **氣**：棋子嘅生命線
- **提子**：點樣食對方嘅棋子
- **禁著點**：唔可以落嘅地方
- **劫**：特殊嘅循環規則
- **眼同活棋**：點樣令棋活落嚟
- **勝負判定**：數目法同數子法

[開始學習圍棋規則 →](./rules)

### 基本術語

了解圍棋專有名詞，等你能夠：

- 睇得明棋譜解說
- 同其他棋友交流
- 理解教學內容

[認識圍棋術語 →](./terminology)

### 對弈禮儀

圍棋唔止係勝負，仲係一種文化。學習：

- 落子嘅禮節
- 對局嘅態度
- 勝負嘅風度

[了解對弈禮儀 →](./etiquette)

### 開局概念

喺正式開始落子之前，先建立正確嘅觀念：

- 佈局嘅大方向
- 角、邊、中嘅價值
- 常見開局位置嘅特性

[學習開局概念 →](./first-10-moves)

## 學習建議

:::tip 畀初學者嘅話
唔好急住想贏棋。啱啱開始學圍棋，最重要嘅係：

1. **確實理解規則** - 特別係「氣」同「眼」嘅概念
2. **多落棋** - 喺 9 路細棋盤上練習，累積經驗
3. **享受過程** - 圍棋係一世嘅興趣，慢慢嚟
:::

## 學習順序

建議按照以下順序學習：

| 順序 | 主題 | 重點 |
|:---:|------|------|
| 1 | 規則 | 先將規則搞清楚 |
| 2 | 術語 | 認識基本用語 |
| 3 | 禮儀 | 培養正確態度 |
| 4 | 開局 | 建立佈局觀念 |
