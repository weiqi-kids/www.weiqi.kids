# Como usar el sitio web Weiqi.Kids

Bienvenido a nuestro sitio web. Aqui encontraras recursos organizados tanto para jugadores de Go como para ingenieros de IA. Puedes acceder a los diferentes temas desde el menu lateral.

## Navegacion por secciones

- **Para jugadores de Go**: Dividido por nivel de habilidad en introduccion, pre-dan y post-dan.
- **Para ingenieros**: Informacion sobre la arquitectura de KataGo, antecedentes de investigacion y codigo fuente.
- **Evolucion del Go**: Historia del desarrollo del Go humano y la trayectoria del Go con IA.
