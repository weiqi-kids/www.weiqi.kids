# مصطلحات Go

تعلم مصطلحات Go يساعدك على فهم السجلات والشروحات والتواصل مع لاعبين آخرين.

## مصطلحات اللوحة

### المواقع المهمة على اللوحة

| المصطلح | الموقع | الشرح |
|------|------|------|
| **تينجن** | مركز اللوحة تماماً | نقطة التقاطع على الخط 10 في لوحة 19×19 (10,10) |
| **هوشي** | النقاط السوداء بالقرب من الزوايا الأربع | الموقع على بعد 4 خطوط من كل حافة (4,4) |
| **كومكو** | نقطة افتتاح شائعة في الزاوية | الموقع (3,4) أو (4,3) |
| **سان-سان** | أعمق نقطة في الزاوية | الموقع (3,3)، طريقة مباشرة لأخذ الزاوية |
| **موكوهازوشي** | موقع أعلى في الزاوية | الموقع (3,5) أو (5,3) |
| **تاكاموكو** | الموقع العالي في الزاوية | الموقع (4,5) أو (5,4) |

### مناطق اللوحة

| المصطلح | الشرح |
|------|------|
| **الزاوية** | مناطق الزوايا الأربع، عادة ضمن الخط الرابع |
| **الحافة** | مناطق الجوانب الأربعة، بين الزوايا |
| **المركز** | منطقة وسط اللوحة، تسمى أيضاً "البطن" |

### ارتفاع المواقع

| المصطلح | الشرح |
|------|------|
| **الخط الأول** | خط الحافة الخارجي (خط الموت) |
| **الخط الثاني** | الخط الثاني (خط الخسارة) |
| **الخط الثالث** | الخط الثالث (خط المنطقة) |
| **الخط الرابع** | الخط الرابع (خط النفوذ) |

:::note مثل شعبي
"الخط الأول للبحث عن الحياة كالتسلق على شجرة، الخط الثاني للخروج قد لا يكفي" - الخطان الأول والثاني مواقع غير مواتية.
:::

## مصطلحات اللعب

### الحركات الأساسية

| المصطلح | الشرح |
|------|------|
| **وضع الحجر** | وضع حجر على اللوحة |
| **الأسر** | إزالة أحجار الخصم التي ليس لها حريات |
| **التضحية** | التخلي عن بعض الأحجار عمداً |

### المصطلحات الهجومية

| المصطلح | الشرح |
|------|------|
| **أتاري** | جعل أحجار الخصم بحرية واحدة فقط |
| **أتاري مزدوج** | تهديد مجموعتين بالأسر في نفس الوقت، الخصم يمكنه إنقاذ واحدة فقط |
| **الشيتشو (السلم)** | أتاري متتالي، الخصم لا يستطيع الهروب |
| **الجيتا (الشبكة)** | حصار فضفاض، حتى لو هرب سيؤسر |

### طرق تحريك الأحجار

| المصطلح | الشرح | الشكل |
|------|------|------|
| **نوبي** | الاتصال في خط مستقيم | امتداد مستقيم |
| **توبي** | القفز بمسافة خط واحد (على نفس الخط) | فراغ واحد بينهما |
| **كيما** | الحركة قطرياً مع فراغ واحد | مثل حركة الفيل في الشطرنج |
| **أوجيما** | حركة قطرية بمسافة أبعد | أبعد من كيما |
| **كوسومي** | الاتصال قطرياً (متجاور قطرياً) | اتصال مائل |
| **إيكن توبي** | القفز بمسافة خط واحد (مثل توبي) | أفقي أو عمودي |

### مصطلحات القتال عن قرب

| المصطلح | الشرح |
|------|------|
| **هاني** | الضغط على زاوية حجر الخصم قطرياً |
| **كيري** | قطع اتصال أحجار الخصم |
| **تسوغي** | وصل أحجارك معاً |
| **أوساي** | منع طريق الخصم |
| **هاسامي** | حصار أحجار الخصم من جانبين |
| **تسوكي** | اللعب ملاصقاً لأحجار الخصم |
| **نوزوكي** | اللعب بجانب حجر الخصم مباشرة |
| **أوشي** | الضغط من الأعلى لجعل الخصم منخفضاً |
| **تسوكي** | الدعم من الأسفل تحت أحجار الخصم |
| **تينوكي** | اللعب في نقطة رئيسية (غالباً في موقع العين) |
| **نوزوكي** | تهديد نقطة ضعف الخصم |

## مصطلحات الأشكال

### الأشكال الجيدة

| المصطلح | الشرح |
|------|------|
| **تورا (النمر)** | حجران قطرياً مع نقطة فارغة بينهما، تشكل فم النمر |
| **الزوج** | الشكل الأساسي بعد اتصال حجرين |
| **أتسوي (السميك)** | شكل بدون نقاط ضعف، مستقر جداً |
| **كاروي (الخفيف)** | شكل مرن، لا يخاف من الهجوم |

### المفاهيم الأساسية

| المصطلح | الشرح |
|------|------|
| **مي (العين)** | نقطة فارغة محاطة بأحجارك |
| **الحريات** | النقاط الفارغة حول الحجر أو المجموعة (خط الحياة) |
| **نقطة القطع** | نقطة ضعف في اتصال الأحجار، يمكن قطعها |
| **الاتصال** | الأحجار المتصلة كوحدة واحدة |

### الأشكال السيئة

| المصطلح | الشرح |
|------|------|
| **جوسوجي (الشكل الأحمق)** | شكل منخفض الكفاءة |
| **دانجو** | أحجار متكدسة معاً، كفاءة منخفضة |
| **كارامي (المثلث الفارغ)** | ثلاثة أحجار تشكل مثلثاً فارغاً، شكل أحمق نموذجي |
| **واريكومي** | شكل منقسم، سهل الهجوم عليه |

## مصطلحات المباراة

### ترتيب الحركات

| المصطلح | الشرح |
|------|------|
| **سينتي** | بعد هذه الحركة، أنت من يقود اللعب |
| **غوتي** | بعد هذه الحركة، تحتاج لترك الخصم يتحرك أولاً |
| **تينوكي** | عدم إكمال الموقف المحلي، والانتقال لمنطقة كبيرة أخرى |
| **سينتي إيكي** | الفائدة المكتسبة من سينتي |

### مراحل المباراة

| المصطلح | الشرح |
|------|------|
| **فوسيكي (الافتتاح)** | المرحلة الأولى، الطرفان يحتلان المناطق الكبيرة |
| **تشوبان (الوسط)** | المرحلة الأكثر حدة في القتال |
| **يوسي (النهاية)** | المرحلة الأخيرة، ترتيب الحدود |
| **يوسي** | إجراء النهاية، تنظيم الحدود |

### تقييم الموقف

| المصطلح | الشرح |
|------|------|
| **يوسي (الميزة)** | الموقف متقدم |
| **فوري (العيب)** | الموقف متأخر |
| **هوسوكي** | موقف بفارق صغير جداً بين الطرفين |
| **تايكيوكو** | الحركة أو الحكم الذي يؤثر على الوضع العام |

### نتائج المباراة

| المصطلح | الشرح |
|------|------|
| **الأسود يفوز بـ X نقطة** | الأسود يفوز بـ X نقطة |
| **الأبيض يفوز بـ X نقطة** | الأبيض يفوز بـ X نقطة |
| **جيغو (التعادل)** | نفس عدد النقاط لكلا الطرفين (نادر جداً) |
| **تشوبان غاتشي** | فوز بدون حساب النقاط (استسلام الخصم) |

## مصطلحات متقدمة

هذه المصطلحات ستواجهها كثيراً في التعلم المتقدم:

| المصطلح | الشرح |
|------|------|
| **جوسيكي** | تسلسلات الحركات المثلى المحددة في الافتتاح |
| **تيسوجي** | حركة ذكية، حركة رائعة |
| **زوكوسوجي** | حركة عادية، منخفضة الكفاءة |
| **نوروتي** | حركة غير عاجلة، تعطي الفائدة للخصم |
| **كيوشو** | موقع مهم يجب اللعب فيه فوراً |
| **أوبا** | موقع ذو قيمة كبيرة |
| **أتسومي** | نفوذ قوي، له تأثير على الخارج |
| **جيتشي** | منطقة محاطة مؤكدة |
| **مويو** | منطقة غير مكتملة لكن لها إمكانات |

:::tip نصيحة تعليمية
لا تحتاج لتذكر كل المصطلحات مرة واحدة. يُنصح بتعلم المصطلحات الأساسية في "مصطلحات اللعب" أولاً، والباقي يمكن تعلمه تدريجياً في الممارسة.
:::
