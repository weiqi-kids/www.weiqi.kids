# गो शब्दावली

गो शब्दावली सीखने से आप गेम रिकॉर्ड समझ सकते हैं, व्याख्या समझ सकते हैं, और अन्य खिलाड़ियों से संवाद कर सकते हैं।

## बोर्ड शब्दावली

### बोर्ड की महत्वपूर्ण स्थितियां

| शब्द | स्थिति | विवरण |
|------|------|------|
| **तेनगेन** | बोर्ड का केंद्र | 19 रेखा बोर्ड का 10वीं रेखा प्रतिच्छेदन (10,10) |
| **स्टार पॉइंट** | चार कोनों के पास के काले बिंदु | किनारे से 4 रेखाएं दूर (4,4) |
| **कोमोकु** | कोने की सामान्य चाल | (3,4) या (4,3) स्थिति |
| **सान-सान** | कोने का सबसे गहरा बिंदु | (3,3) स्थिति, सीधे कोना लेने की चाल |
| **मोकुहाजुशी** | कोने की ऊंची स्थिति | (3,5) या (5,3) स्थिति |
| **ताकामोकु** | कोने की उच्च स्थिति | (4,5) या (5,4) स्थिति |

### बोर्ड के क्षेत्र

| शब्द | विवरण |
|------|------|
| **कोना** | बोर्ड के चार कोने, आमतौर पर 4 रेखा के अंदर |
| **किनारा** | बोर्ड के चार किनारे, कोनों के बीच का भाग |
| **केंद्र** | बोर्ड का मध्य क्षेत्र |

### स्थिति की ऊंचाई

| शब्द | विवरण |
|------|------|
| **पहली रेखा** | सबसे बाहरी किनारा (मृत्यु रेखा) |
| **दूसरी रेखा** | दूसरी रेखा (हार रेखा) |
| **तीसरी रेखा** | तीसरी रेखा (क्षेत्र रेखा) |
| **चौथी रेखा** | चौथी रेखा (प्रभाव रेखा) |

:::note कहावत
"पहली रेखा पर जीवन खोजना मछली की तरह, दूसरी रेखा से निकलना घाटे में" - पहली और दूसरी रेखा प्रतिकूल स्थितियां हैं।
:::

## खेल शब्दावली

### बुनियादी क्रियाएं

| शब्द | विवरण |
|------|------|
| **चाल** | पत्थर को बोर्ड पर रखना |
| **कैप्चर** | बिना लिबर्टी के प्रतिद्वंद्वी के पत्थर हटाना |
| **त्याग** | जानबूझकर कुछ पत्थर नहीं बचाना |

### हमला संबंधित

| शब्द | विवरण |
|------|------|
| **अटारी** | प्रतिद्वंद्वी के पत्थर में सिर्फ एक लिबर्टी छोड़ना |
| **डबल अटारी** | एक साथ दो समूहों पर अटारी, प्रतिद्वंद्वी एक ही बचा सकता है |
| **लैडर** | लगातार अटारी, प्रतिद्वंद्वी भाग नहीं सकता |
| **नेट** | ढीले से घेरना, भागने पर भी पकड़ा जाएगा |

### पत्थर की चाल के तरीके

| शब्द | विवरण | आकार |
|------|------|------|
| **एक्सटेंड** | सीधी रेखा में जोड़ना | सीधा विस्तार |
| **जंप** | एक खाना छोड़कर (एक ही रेखा पर) | बीच में एक खाली |
| **नाइट्स मूव** | विकर्ण दिशा में, बीच में एक खाली | शतरंज के घोड़े जैसा |
| **लार्ज नाइट्स मूव** | विकर्ण दिशा, अधिक दूरी | नाइट्स मूव से दूर |
| **डायगोनल** | विकर्ण रूप से सटा (तिरछा आसन्न) | तिरछा जोड़ |
| **वन-पॉइंट जंप** | एक ही रेखा पर एक खाना छोड़कर | क्षैतिज या ऊर्ध्वाधर |

### संपर्क युक्तियां

| शब्द | विवरण |
|------|------|
| **हेन** | प्रतिद्वंद्वी के पत्थर के तिरछे कोने पर रोकना |
| **कट** | प्रतिद्वंद्वी के पत्थरों का जोड़ तोड़ना |
| **कनेक्ट** | अपने पत्थरों को जोड़ना |
| **ब्लॉक** | प्रतिद्वंद्वी का रास्ता रोकना |
| **पिंसर** | दोनों तरफ से प्रतिद्वंद्वी को घेरना |
| **अटैच** | प्रतिद्वंद्वी के पत्थर से सटकर खेलना |
| **प्रेस** | ऊपर से दबाना, प्रतिद्वंद्वी को नीचे करना |
| **अंडरकट** | प्रतिद्वंद्वी के पत्थर के नीचे सहारा देना |
| **प्लेसमेंट** | महत्वपूर्ण स्थान पर एक पत्थर (अक्सर आंख या मुख्य बिंदु पर) |
| **पीप** | प्रतिद्वंद्वी की कमजोरी पर धमकी देना |

## आकार शब्दावली

### अच्छे आकार

| शब्द | विवरण |
|------|------|
| **टाइगर्स माउथ** | दो पत्थर तिरछे, बीच में खाली, बाघ का मुंह बनाता है |
| **बैंबू जॉइंट** | दो पत्थर जुड़ने के बाद का बुनियादी आकार |
| **मोटा** | कोई कमजोरी नहीं, बहुत स्थिर |
| **हल्का** | लचीला आकार, हमले से नहीं डरता |

### बुनियादी अवधारणाएं

| शब्द | विवरण |
|------|------|
| **आंख** | अपने पत्थरों से घिरा खाली बिंदु |
| **लिबर्टी** | पत्थर के आसपास के खाली बिंदु (जीवन रेखा) |
| **कटिंग पॉइंट** | पत्थर जोड़ की कमजोरी, जहां काटा जा सकता है |
| **कनेक्शन** | पत्थर एक इकाई बनकर जुड़े |

### खराब आकार

| शब्द | विवरण |
|------|------|
| **बेवकूफ आकार** | कम कुशल आकार |
| **भारी आकार** | पत्थर एक साथ जमे, कुशलता कम |
| **खाली त्रिकोण** | तीन पत्थर खाली त्रिकोण बनाते, विशिष्ट बेवकूफ आकार |
| **विभाजित आकार** | आकार अलग, हमले में आसान |

## मैच शब्दावली

### चाल का क्रम

| शब्द | विवरण |
|------|------|
| **सेंते** | यह चाल खेलने के बाद, आप नियंत्रण में |
| **गोते** | यह चाल खेलने के बाद, प्रतिद्वंद्वी को पहले खेलने देना |
| **टेनुकी** | स्थानीय रूप से समाप्त न करते हुए, अन्य बड़े बिंदु पर जाना |
| **सेंते लाभ** | सेंते से मिला फायदा |

### मैच चरण

| शब्द | विवरण |
|------|------|
| **ओपनिंग** | शुरुआती चरण, दोनों बड़े बिंदु लेते हैं |
| **मिडल गेम** | सबसे तीव्र लड़ाई का चरण |
| **एंडगेम** | अंतिम चरण, सीमाएं निपटाना |
| **समापन** | एंडगेम करना, सीमाएं साफ करना |

### स्थिति मूल्यांकन

| शब्द | विवरण |
|------|------|
| **लाभ** | आगे की स्थिति |
| **हानि** | पीछे की स्थिति |
| **करीबी खेल** | दोनों का अंतर बहुत कम |
| **बड़ी तस्वीर** | पूरी स्थिति को प्रभावित करने वाली चाल या निर्णय |

### मैच परिणाम

| शब्द | विवरण |
|------|------|
| **काला X अंक से जीता** | काला X अंक से जीता |
| **सफेद X अंक से जीता** | सफेद X अंक से जीता |
| **ड्रॉ** | दोनों के अंक समान (बहुत दुर्लभ) |
| **मिड-गेम जीत** | अंक गिने बिना जीत (प्रतिद्वंद्वी ने हार मानी) |

## उन्नत शब्दावली

उन्नत सीखने में ये शब्द अक्सर मिलेंगे:

| शब्द | विवरण |
|------|------|
| **जोसेकी** | ओपनिंग में दोनों की सर्वोत्तम प्रतिक्रिया की निश्चित चाल |
| **तेसुजी** | चतुर तकनीक, शानदार चाल |
| **साधारण चाल** | सामान्य, कम कुशल चाल |
| **धीमी चाल** | जरूरी नहीं, प्रतिद्वंद्वी को फायदा देती |
| **महत्वपूर्ण बिंदु** | तुरंत खेलना जरूरी |
| **बड़ा बिंदु** | उच्च मूल्य की स्थिति |
| **मोटा प्रभाव** | मजबूत प्रभाव, बाहर पर असर |
| **क्षेत्र** | निश्चित घेरा हुआ इलाका |
| **मोयो** | अभी पूरा नहीं पर संभावित इलाका |

:::tip सीखने का सुझाव
सभी शब्द एक बार में याद करने की जरूरत नहीं। पहले "खेल शब्दावली" की बुनियादी बातें जानें, बाकी वास्तविक खेल में धीरे-धीरे सीखें।
:::
