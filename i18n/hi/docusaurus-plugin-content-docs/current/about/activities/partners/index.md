# पार्टनर्स

निम्नलिखित संगठनों और व्यक्तियों का धन्यवाद जो गो संस्कृति को बढ़ावा देने के लिए हमारे साथ काम करते हैं।

| पार्टनर | विशेषज्ञता | सहयोग |
|---------|-----------|--------|
| [वू डी](./wu-di) | इलस्ट्रेटर | कैरेक्टर डिज़ाइन (डोरा, याबो, ड्रैगन, टाइगर) |
| [माओ ज़ी](./mao-zi) | इलस्ट्रेटर | गो थीम वॉलपेपर |
