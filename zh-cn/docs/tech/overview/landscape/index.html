圍棋 AI 生態全景圖

圍棋 AI 不只是單一的程式，而是一個完整的生態系統。這篇文章將幫助你了解各種選擇，找到最適合你的工具。

選擇建議​

你是...

工程師/開發者 → KataGo（最強、開源、API 完整）
研究者 → KataGo 或 ELF OpenGo（可訓練）
一般使用者 → KataGo + KaTrain（介面友善）
職業棋手 → 所屬棋院的官方 AI

技術架構圖​

KataGo 模型選擇​

KataGo 提供多種模型，需根據硬體和需求選擇：

模型檔案大小棋力速度適用硬體
b10c128~20 MB中等最快低階硬體、快速測試
b18c384~140 MB強快中階 GPU
b40c256~250 MB很強中高階 GPU
b60c320~500 MB最強慢頂級 GPU

選擇建議：

  • 一般使用：b18c384（平衡）
  • 快速分析：b10c128
  • 最高品質：b60c320
