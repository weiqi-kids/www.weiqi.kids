# 趙珍

**深圳五味中醫館 中醫科 主治醫師**

## 關於

趙珍是一位中醫科主治醫師，畢業於遼寧中醫藥大學，師從山東名醫門延松，現任職於深圳五味中醫館。臨床擅長針藥結合治療，對疼痛證與慢性病調理有一定見解。內容創作以中醫觀念解釋常見健康問題，強調體質調理、生活方式調整與長期健康管理，幫助讀者以更容易理解的方式認識中醫調理思路。

## 專業領域

- 婦科疾病調理（月經不調、痛經、不孕、帶下病、產後調理）
- 脾胃疾病調理（胃痛、胃脹、便秘、呃逆）
- 皮膚疾病調理（蕁麻疹、濕疹、青春痘、黃褐斑）
- 常見內科與體質調理（失眠、頭痛、眩暈、高血壓、糖尿病）
- 針藥結合治療（肥胖、失眠、痔瘡、各類疼痛症）
