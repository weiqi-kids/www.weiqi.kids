# 郭尚諺 Ian

## 關於

郭尚諺是一位擁有 14 年音樂經歷的古典吉他演奏家與數位音樂創作者。他主修心理學，將對人心的理解融入音樂創作中，專精於商業音樂製作。

## 專業領域

- 古典吉他演出
- 數位音樂創作
- 商業音樂製作
- 影片剪輯製作

## 學歷
