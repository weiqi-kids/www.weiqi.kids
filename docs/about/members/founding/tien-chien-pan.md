# 潘天健

![潘天健](/img/members/tien-chien-pan.png)

**沁安整合醫學診所 院長**

## 關於

潘天健醫師是一位兼具西醫內科專科執照的中醫師，致力於結合中醫、西醫、功能醫學和其他醫療方式，提供整合性的健康照護服務。

## 現任職務

- 沁安整合醫學診所 院長
- 台灣細胞分子矯正學會 理事長
- 天健國際生醫股份有限公司 董事長
