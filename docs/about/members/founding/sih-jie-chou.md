# 周思傑

![周思傑](/img/members/sih-jie-chou.png)

**眾城國際法律事務所 律師**

## 關於

周思傑是擁有 18 年執業經驗的資深律師，國立臺灣大學法律系畢業、東海大學法律研究所碩士班公法組。他大學四年級即通過地政士考試，公務人員高考財稅法務類組錄取、政風類組臺北市第三名錄取，行政院公平交易法研習班第 40 期結業。事務所主打「全方位企業法律顧問 + 訴訟案件優質處理 + 資產傳承合法規劃」，並把「保護權益的契約審查」與「法律風險控管」當作核心服務。

## 服務項目

1. **全方位企業法律顧問**：法規遵循、決策風險評估、契約、貨款、人事
2. **訴訟案件優質處理**：民事／行政訴訟代理人、刑事告訴代理人、辯護人
3. **資產傳承避免後續爭議的合法規劃**（40-70 世代 + 1 名以上繼承人）
4. **保護權益的契約審查**
5. **法律風險控管**
