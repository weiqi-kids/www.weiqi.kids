# 周思傑

## 關於

周思傑律師是一位擁有 18 年執業經驗的資深律師，畢業於國立台灣大學法律學系。他的專業涵蓋民刑事訴訟、企業法務及各類商業糾紛處理。

## 專業領域

- 民事訴訟、刑事訴訟
- 行政訴訟
- 公司法、投資法律
- 勞動法律
- 企業併購
- 保險法規、銀行法規
- 不動產糾紛

## 學歷
