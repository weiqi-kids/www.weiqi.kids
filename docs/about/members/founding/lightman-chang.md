# 張饒輝 CΛ / Lightman

## 角色定位

不只是技術工作者，也不只是內容創作者，而是站在 **問題分析、根因追查、方法設計、系統整合與知識轉譯** 交界處的整合者。長期關注 AI、數位健康、醫療資訊整合、產品設計與內容轉譯，同時投入圍棋教育、思考訓練與知識傳播。

寫作與思考核心是「**問題、原因、方法、最好**」四個字 — 先把真正的問題定義清楚，再往下追出形成問題的原因，提出可執行的方法，最後比較什麼做法更好，並盡量把問題從源頭防範於未然。

## 寫作風格

**清楚、務實、好懂；有系統感與架構感；重視問題定義與根本原因；不浮誇、不空泛、不說教**。內容定位適合作為中立、結構化、重邏輯、重根因、重實用性的社論或專欄。

### 表達邏輯

1. 先定義真正的問題
2. 再拆出造成問題的原因
3. 再整理可執行的方法
4. 最後比較什麼做法更好，並回到如何從根本預防問題再次發生
