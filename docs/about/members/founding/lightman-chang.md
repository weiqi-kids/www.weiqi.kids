# 張饒輝 Lightman Chang

**台灣好棋寶寶協會 理事長**

## 關於

張饒輝是台灣好棋寶寶協會的創辦人暨理事長。他扮演協會「AI 整合者」角色，將跨域夥伴的領域知識整合為開源 AI 工具，把產業 know-how 轉化為可被社會使用的公益產出。本業跨足科技、圍棋教育與醫療科技三個面向，並透過廣東省圍棋初級教師資格認證，將圍棋教學能力從本地擴展到華語圍棋培訓系統。
