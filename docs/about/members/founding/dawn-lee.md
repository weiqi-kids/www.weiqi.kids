## 關於

李郁萱是擁有醫療器材碩士與多年品保主管背景的 ISO 顧問師。她從醫療器材公司品保主管轉任企業 ISO 系統輔導，把「規劃可長久使用的管理系統」當成本業，協助企業在品質、食安、資安、職業安全衛生、環境、碳足跡、社會責任等面向取得國際標準認證。

## 做 ISO 的價值

1. 提高工作效率、生產力與利潤
2. 讓內部管理溝通順暢、更有效率
3. 維持產品與服務的穩定性
4. 建立優化與改善的機制
