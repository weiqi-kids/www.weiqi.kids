# 謝依純

**專業配音員**

## 關於

謝依純是一位專業配音員，為好棋寶寶協會提供宣傳影片的配音服務。她的聲音作品涵蓋廣告、企業宣傳、動畫配音等多種類型。

## 配音作品集

謝依純曾為多家知名品牌配音，包括：

- 南山人壽 商品篇
- 康是美 Saeve 上市宣傳
- 痞客邦 企業宣傳
- 希爾思 寵物食品廣告
- 麥當勞 冷飲廣告
- 韓劇《又是吳海英》配音

## 合作內容

為好棋寶寶協會的宣傳影片、活動介紹提供專業配音服務，以溫暖親切的聲音傳達圍棋文化的魅力。

## 服務項目

- 企業宣傳配音
- 活動介紹旁白
- 動畫角色配音
- 廣告配音
