## 學習路徑建議

### 路徑 A：理解原始碼

```
原始碼導讀 → 閱讀核心模組 → 嘗試修改 → 提交貢獻
```

適合：
- 想了解工業級 AI 系統架構
- 想學習 C++ 高效能程式設計
- 想貢獻程式碼

### 路徑 B：KataGo 訓練機制解析

```
訓練指南 → 設定環境 → 小規模實驗 → 調整參數 → 完整訓練
```

適合：
- 想理解深度學習訓練流程
- 想實驗不同的網路架構
- 想研究強化學習

### 路徑 C：參與社群

```
了解社群 → 設定分散式訓練 → 貢獻算力 → 參與討論
```

適合：
- 想參與開源專案
- 想了解分散式訓練
- 想與全球開發者交流

## 硬體需求

### 閱讀原始碼

無特殊需求，任何電腦都可以。

### 訓練模型

| 規模 | 建議硬體 | 訓練時間 |
|------|---------|---------|
| 迷你（b6c96） | GTX 1060 6GB | 數小時 |
| 小型（b10c128） | RTX 3060 12GB | 1-2 天 |
| 中型（b18c384） | RTX 4090 24GB | 1-2 週 |
| 完整（b40c256） | 多 GPU 集群 | 數週 |

### 分散式訓練貢獻

- 任何有 GPU 的電腦都可以參與
- 建議至少 GTX 1060 或同等級
- 需要穩定的網路連線
