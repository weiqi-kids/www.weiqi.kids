# 圍棋 AI 能做什麼？

在深入技術細節之前，先來看看現代圍棋 AI 能做到什麼。這些功能將幫助你理解為什麼圍棋 AI 如此重要，以及它在實際應用中的價值。

## 核心功能展示

### 1. 即時勝率預測

AI 可以在每一步棋後即時計算雙方的勝率。

```
當前局面：黑方勝率 62.3%
          白方勝率 37.7%

上一手影響：黑方勝率 +5.2%（好棋）
```

**技術原理**：Value Network 輸入棋盤狀態，輸出一個 -1 到 +1 的數值，代表當前局面的勝率評估。

### 2. 目數預測

不只知道誰贏，還能預測贏多少目。

```
預測結果：黑方領先 7.5 目
信賴區間：5.5 - 9.5 目（95%）
```

**技術原理**：KataGo 的多頭網路設計，同時輸出勝率與目數預測，提供更精確的形勢判斷。

### 3. 領地估算

視覺化顯示每個位置最終歸屬的機率。

```
領地分布圖：
■ 黑方確定領地（>90%）
▪ 黑方傾向領地（60-90%）
· 雙方爭奪區域
▫ 白方傾向領地（60-90%）
□ 白方確定領地（>90%）
```

**技術原理**：Ownership Head 預測每個位置在終局時歸屬黑方或白方的機率。

### 4. 候選下法分析

顯示 AI 認為最好的幾個下法，以及每個下法的詳細資訊。

| 排名 | 位置 | 勝率 | 目數 | 訪問次數 |
|------|------|------|------|---------|
| 1 | Q16 | 63.5% | +7.2 | 1,234 |
| 2 | R14 | 62.8% | +6.8 | 856 |
| 3 | D4 | 62.1% | +6.5 | 423 |

**技術原理**：MCTS 搜索過程中，訪問次數最多的節點通常是最佳選擇。

### 5. 覆盤輔助

自動標記重大失誤手，並提供改進建議。

```
第 42 手：❌ 失誤手
- 實戰下法：P10
- 勝率變化：-15.3%
- AI 建議：Q12（維持 62% 勝率）
- 最佳變化圖：Q12 → R13 → Q13 → ...
```

**技術原理**：比較每手棋前後的勝率變化，超過閾值（如 5%）的下降標記為失誤。

### 教學輔助

- **自動難度調整**：根據學生棋力調整 AI 強度
- **即時提示**：在關鍵位置給予提示
- **錯誤診斷**：自動識別常見錯誤模式

### 棋譜研究

- **大量棋譜批次分析**：自動標記所有失誤手
- **風格分析**：比較不同棋手的下棋風格
- **開局研究**：分析定式的優劣
