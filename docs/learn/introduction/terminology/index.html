圍棋術語

學習圍棋術語，能幫助你看懂棋譜、理解解說，並與其他棋友交流。

棋盤術語​

棋盤的重要位置​

術語位置說明
天元棋盤正中央19 路棋盤的第 10 線交點（10,10）
星位四個角落附近的黑點距離邊線各 4 條線的位置（4,4）
小目角落的常見落子點(3,4) 或 (4,3) 的位置
三三角落最深處(3,3) 的位置，直接取角的下法
目外角落較高的位置(3,5) 或 (5,3) 的位置
高目角落的高位(4,5) 或 (5,4) 的位置

棋盤的區域​

術語說明
角棋盤四個角落區域，通常指 4 線以內
邊棋盤四邊區域，角落之間的部分
中腹棋盤中央區域，也稱「中央」或「腹地」

位置的高低​

術語說明
一線最外面的邊線（死亡線）
二線第二條線（敗線）
三線第三條線（實地線）
四線第四條線（勢力線）
俗諺

「一線求活緣木魚，二線出頭恐不盈」—— 一、二線是不利的位置。

下棋術語​

基本動作​

術語說明
落子將棋子放到棋盤上
提子把沒有氣的對方棋子拿走
棄子故意放棄某些棋子不救

攻擊相關​

術語說明
打吃 / 叫吃讓對方棋子只剩一口氣
雙叫吃同時叫吃兩塊棋，對方只能救一塊
征子（征吃）連續叫吃，對方逃不出的吃法
枷寬鬆地封住對方，跑也會被吃

棋子的移動方式​

術語說明形狀
長沿著直線連接出去直線延伸
跳隔一格跳出（同線上）中間空一格
飛對角方向移動，中間空一格類似象棋的象
大飛對角方向移動，距離更遠比飛更遠
尖對角緊靠（斜向相鄰）斜著連接
關同線上隔一格跳（同跳）橫向或縱向

接觸戰術語​

術語說明
扳在對方棋子斜角頂住
斷切斷對方棋子的連接
接把自己的棋子連接起來
擋擋住對方的去路
夾從兩邊夾住對方的棋子
靠緊貼著對方的棋子下
貼緊挨著下，通常在對方棋子旁邊
壓從上方壓住對方，讓對方變低
托在對方棋子下方托住
點在關鍵位置下一子（常在眼位或要點）
刺往對方的弱點下一子威脅

棋形術語​

好形​

術語說明
虎兩子對角，中間空一點，形成虎口
雙兩子連接後的基本形狀
厚實棋形沒有弱點，非常穩固
輕靈棋形有彈性，不怕被攻

基本概念​

術語說明
眼被己方棋子圍住的空點
氣棋子或棋塊周圍的空點（生命線）
斷點棋子連接的弱點，可能被切斷的地方
連接棋子相連形成一體

不好的形​

術語說明
愚形效率差的棋形
凝形棋子擠在一起，效率不高
空三角三子圍成空三角，典型的愚形
裂形棋形被分開，容易被攻擊

對局術語​

手的順序​

術語說明
先手下完這手後，輪到我方繼續主導
後手下完這手後，需要讓對方先動
搶先在局部不收完，轉向其他大場
先手利先手得到的便宜

對局階段​

術語說明
佈局開局階段，雙方佔領大場
中盤戰鬥最激烈的階段
官子終局階段，收拾邊界
收官進行官子，把邊界收好

局面判斷​

術語說明
優勢局面領先
劣勢局面落後
細棋雙方差距很小的局面
大局影響整體局勢的下法或判斷

對局結果​

術語說明
黑勝 X 目黑方贏 X 目
白勝 X 目白方贏 X 目
和棋雙方目數相同（非常罕見）
中盤勝不計算目數的勝利（對方認輸或棄權）

進階術語​

這些術語在進階學習時會經常遇到：

術語說明
定式開局時雙方最佳應對的固定下法
手筋巧妙的手法，精彩的一手
俗手普通、效率不高的下法
緩著不夠緊迫、讓對方得利的下法
急所必須馬上下的重要位置
大場價值很大的位置
厚勢厚實的勢力，對外有影響力
實地確定圍住的地盤
模樣尚未完成但有潛力的地盤
學習建議

不需要一次記住所有術語。建議先熟悉「下棋術語」中的基本用詞，其他的在實戰中慢慢學習即可。
