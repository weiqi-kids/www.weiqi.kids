# 圍棋術語

學習圍棋術語，能幫助你看懂棋譜、理解解說，並與其他棋友交流。

## 棋盤術語

### 棋盤的重要位置

| 術語 | 位置 | 說明 |
|------|------|------|
| **天元** | 棋盤正中央 | 19 路棋盤的第 10 線交點（10,10） |
| **星位** | 四個角落附近的黑點 | 距離邊線各 4 條線的位置（4,4） |
| **小目** | 角落的常見落子點 | (3,4) 或 (4,3) 的位置 |
| **三三** | 角落最深處 | (3,3) 的位置，直接取角的下法 |
| **目外** | 角落較高的位置 | (3,5) 或 (5,3) 的位置 |
| **高目** | 角落的高位 | (4,5) 或 (5,4) 的位置 |

### 棋盤的區域

| 術語 | 說明 |
|------|------|
| **角** | 棋盤四個角落區域，通常指 4 線以內 |
| **邊** | 棋盤四邊區域，角落之間的部分 |
| **中腹** | 棋盤中央區域，也稱「中央」或「腹地」 |

### 位置的高低

| 術語 | 說明 |
|------|------|
| **一線** | 最外面的邊線（死亡線） |
| **二線** | 第二條線（敗線） |
| **三線** | 第三條線（實地線） |
| **四線** | 第四條線（勢力線） |

:::note 俗諺
「一線求活緣木魚，二線出頭恐不盈」—— 一、二線是不利的位置。
:::

## 下棋術語

### 基本動作

| 術語 | 說明 |
|------|------|
| **落子** | 將棋子放到棋盤上 |
| **提子** | 把沒有氣的對方棋子拿走 |
| **棄子** | 故意放棄某些棋子不救 |

### 攻擊相關

| 術語 | 說明 |
|------|------|
| **打吃 / 叫吃** | 讓對方棋子只剩一口氣 |
| **雙叫吃** | 同時叫吃兩塊棋，對方只能救一塊 |
| **征子（征吃）** | 連續叫吃，對方逃不出的吃法 |
| **枷** | 寬鬆地封住對方，跑也會被吃 |

### 棋子的移動方式

| 術語 | 說明 | 形狀 |
|------|------|------|
| **長** | 沿著直線連接出去 | 直線延伸 |
| **跳** | 隔一格跳出（同線上） | 中間空一格 |
| **飛** | 對角方向移動，中間空一格 | 類似象棋的象 |
| **大飛** | 對角方向移動，距離更遠 | 比飛更遠 |
| **尖** | 對角緊靠（斜向相鄰） | 斜著連接 |
| **關** | 同線上隔一格跳（同跳） | 橫向或縱向 |

### 接觸戰術語

| 術語 | 說明 |
|------|------|
| **扳** | 在對方棋子斜角頂住 |
| **斷** | 切斷對方棋子的連接 |
| **接** | 把自己的棋子連接起來 |
| **擋** | 擋住對方的去路 |
| **夾** | 從兩邊夾住對方的棋子 |
| **靠** | 緊貼著對方的棋子下 |
| **貼** | 緊挨著下，通常在對方棋子旁邊 |
| **壓** | 從上方壓住對方，讓對方變低 |
| **托** | 在對方棋子下方托住 |
| **點** | 在關鍵位置下一子（常在眼位或要點） |
| **刺** | 往對方的弱點下一子威脅 |

## 棋形術語

### 好形

| 術語 | 說明 |
|------|------|
| **虎** | 兩子對角，中間空一點，形成虎口 |
| **雙** | 兩子連接後的基本形狀 |
| **厚實** | 棋形沒有弱點，非常穩固 |
| **輕靈** | 棋形有彈性，不怕被攻 |

### 基本概念

| 術語 | 說明 |
|------|------|
| **眼** | 被己方棋子圍住的空點 |
| **氣** | 棋子或棋塊周圍的空點（生命線） |
| **斷點** | 棋子連接的弱點，可能被切斷的地方 |
| **連接** | 棋子相連形成一體 |

### 不好的形

| 術語 | 說明 |
|------|------|
| **愚形** | 效率差的棋形 |
| **凝形** | 棋子擠在一起，效率不高 |
| **空三角** | 三子圍成空三角，典型的愚形 |
| **裂形** | 棋形被分開，容易被攻擊 |

## 對局術語

### 手的順序

| 術語 | 說明 |
|------|------|
| **先手** | 下完這手後，輪到我方繼續主導 |
| **後手** | 下完這手後，需要讓對方先動 |
| **搶先** | 在局部不收完，轉向其他大場 |
| **先手利** | 先手得到的便宜 |

### 對局階段

| 術語 | 說明 |
|------|------|
| **佈局** | 開局階段，雙方佔領大場 |
| **中盤** | 戰鬥最激烈的階段 |
| **官子** | 終局階段，收拾邊界 |
| **收官** | 進行官子，把邊界收好 |

### 局面判斷

| 術語 | 說明 |
|------|------|
| **優勢** | 局面領先 |
| **劣勢** | 局面落後 |
| **細棋** | 雙方差距很小的局面 |
| **大局** | 影響整體局勢的下法或判斷 |

### 對局結果

| 術語 | 說明 |
|------|------|
| **黑勝 X 目** | 黑方贏 X 目 |
| **白勝 X 目** | 白方贏 X 目 |
| **和棋** | 雙方目數相同（非常罕見） |
| **中盤勝** | 不計算目數的勝利（對方認輸或棄權） |

## 進階術語

這些術語在進階學習時會經常遇到：

| 術語 | 說明 |
|------|------|
| **定式** | 開局時雙方最佳應對的固定下法 |
| **手筋** | 巧妙的手法，精彩的一手 |
| **俗手** | 普通、效率不高的下法 |
| **緩著** | 不夠緊迫、讓對方得利的下法 |
| **急所** | 必須馬上下的重要位置 |
| **大場** | 價值很大的位置 |
| **厚勢** | 厚實的勢力，對外有影響力 |
| **實地** | 確定圍住的地盤 |
| **模樣** | 尚未完成但有潛力的地盤 |

:::tip 學習建議
不需要一次記住所有術語。建議先熟悉「下棋術語」中的基本用詞，其他的在實戰中慢慢學習即可。
:::
