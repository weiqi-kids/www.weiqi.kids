# 入門篇

歡迎踏入圍棋的世界！這個區塊將帶你從零開始，建立扎實的圍棋基礎。

## 本區塊內容

### 圍棋規則

圍棋的規則簡單卻深邃。你將學習：

- **氣**：棋子的生命線
- **提子**：如何吃掉對方的棋子
- **禁著點**：不能下的地方
- **劫**：特殊的循環規則
- **眼與活棋**：如何讓棋活下來
- **勝負判定**：數目法與數子法

[開始學習圍棋規則 →](./rules)

### 基本術語

了解圍棋專有名詞，讓你能夠：

- 看懂棋譜解說
- 與其他棋友交流
- 理解教學內容

[認識圍棋術語 →](./terminology)

### 對弈禮儀

圍棋不只是勝負，更是一種文化。學習：

- 落子的禮節
- 對局的態度
- 勝負的風度

[了解對弈禮儀 →](./etiquette)

### 開局概念

在正式開始下棋之前，先建立正確的觀念：

- 佈局的大方向
- 角、邊、中的價值
- 常見開局位置的特性

[學習開局概念 →](./first-10-moves)

## 學習建議

:::tip 給初學者的話
不要急著想贏棋。剛開始學圍棋，最重要的是：

1. **確實理解規則** - 特別是「氣」和「眼」的概念
2. **多下棋** - 在 9 路小棋盤上練習，累積經驗
3. **享受過程** - 圍棋是一輩子的興趣，慢慢來
:::

## 學習順序

建議按照以下順序學習：

| 順序 | 主題 | 重點 |
|:---:|------|------|
| 1 | 規則 | 先把規則弄清楚 |
| 2 | 術語 | 認識基本用語 |
| 3 | 禮儀 | 培養正確態度 |
| 4 | 開局 | 建立佈局觀念 |
