圍棋規則

圍棋嘅規則非常簡單，但由此產生嘅變化卻無窮無盡。呢個正正係圍棋迷人嘅地方。

基本概念​

棋盤同棋子​

  • 棋盤：標準棋盤係 19 路（19×19 條線），初學者常用 9 路或 13 路
  • 棋子：黑白兩色，黑子先落
  • 落子位置：棋子落喺線嘅交叉點上，唔係格仔入面

遊戲目標​

圍棋嘅目標係圍地。終局時，圍住較多空地嗰方獲勝。

氣嘅概念​

氣係圍棋最重要嘅概念。氣係棋子周圍空嘅交叉點，係棋子嘅「生命線」。

一粒棋子嘅氣​

一粒棋子喺唔同位置，氣嘅數量唔同：

位置氣嘅數量
中央4 氣
邊上3 氣
角上2 氣

連接嘅棋子​

當同色棋子相鄰（上下左右連接）嗰陣，佢哋會成為一個整體，共享所有嘅氣。

重要觀念

相連嘅棋子同生共死。如果呢塊棋被食，所有連接嘅棋子都會一齊被提走。

提子​

當一塊棋嘅氣被對方全部堵住（氣為 0）嗰陣，呢塊棋就被「提掉」，由棋盤上攞走。

提子嘅步驟​

  1. 對方嘅棋子淨返一口氣
  2. 你落一手堵住最後一口氣
  3. 對方嘅棋子被提走（攞離棋盤）

叫吃（打吃）​

當一塊棋淨返一口氣嗰陣，呢個狀態叫做叫吃（或打吃）。呢個時候對方必須設法走甩或者棄子。

禁著點（禁入點）​

有啲位置係唔可以落嘅，呢啲叫做禁著點或禁入點。

判斷禁著點​

一個位置係禁著點，必須同時滿足：

  1. 落喺嗰度之後，自己嘅棋子冇氣
  2. 而且冇辦法提掉任何對方嘅棋子
簡單記法

如果落子之後能夠提掉對方嘅棋，就唔係禁著點。

自殺手​

落一手令自己嘅棋子冇氣，又冇辦法提掉對方嘅棋，呢個叫「自殺」。圍棋規則禁止自殺。

劫嘅規則​

劫係圍棋入面一種特殊嘅形狀，會造成無限循環嘅局面。

咩係劫​

當雙方可以互相提一子，而且提完之後對方都可以即刻提返，就形成咗劫。

劫嘅規則​

唔可以即刻提返。被提劫之後，必須先喺其他地方落一手（叫做「搵劫材」），然後先可以提返。

打劫嘅過程​

  1. 甲方提劫
  2. 乙方唔可以即刻提返，必須先喺其他地方落
  3. 甲方應對乙方呢手棋
  4. 乙方提返劫
  5. 如此往復，直到一方放棄
點解有呢條規則

如果冇劫嘅規則，雙方可以無限循環互提，棋局永遠冇辦法結束。

眼同活棋​

眼係圍棋入面最重要嘅概念之一。理解眼，就理解咗棋嘅死活。

咩係眼​

眼係被自己棋子完全包圍嘅空點。對方冇辦法喺眼位落子（會係禁著點）。

活棋嘅條件​

一塊棋要活，必須有兩個或以上嘅真眼。

點解需要兩個眼？

  • 如果淨係得一個眼，對方可以由外圍逐漸緊氣
  • 最後呢個眼就係最後一口氣，對方可以落入去食咗成塊棋
  • 有兩個眼嗰陣，對方冇辦法同時佔據兩個眼位，呢塊棋就永遠食唔到

真眼同假眼​

  • 真眼：完整嘅眼，對方冇辦法破壞
  • 假眼：睇落似眼，但有缺陷，可能被對方破壞

判斷真假眼需要睇眼位嘅對角線位置，呢個係進階嘅死活知識。

勝負判定​

對局結束嗰陣，需要計算雙方嘅地嚟判定勝負。主要有兩種計算方法。

數目法（日韓規則）​

計算雙方圍住嘅空點數量。

計算方式：

  • 己方圍住嘅空點（目）
  • 唔計算己方棋盤上嘅棋子

勝負判定：目數多嗰方獲勝。

數子法（中國規則）​

計算己方嘅子同地嘅總數。

計算方式：

  • 己方棋盤上嘅活棋數量（子）
  • 加上己方圍住嘅空點（目）

勝負判定：

  • 標準棋盤共 361 個交叉點
  • 超過 180.5 點嗰方獲勝

貼目（讓先）​

因為黑棋先落有優勢，白棋會得到補償嘅目數，叫做「貼目」。

規則貼目
中國規則黑貼 3.75 子（7.5 目）
日本規則黑貼 6.5 目
韓國規則黑貼 6.5 目
初學者建議

啱啱開始學棋嗰陣唔使太在意勝負判定嘅細節。先將「氣」同「眼」嘅概念搞清楚，呢個先係最重要嘅基礎。

終局​

幾時結束​

當雙方都認為冇可以落嘅地方嗰陣，對局結束。實際上係雙方連續虛手（pass）。

終局處理​

  1. 確認所有死棋
  2. 將死棋由棋盤上攞走
  3. 計算雙方嘅地
  4. 根據規則判定勝負

認輸​

棋局進行中，如果一方認為已經冇獲勝嘅可能，可以隨時認輸。認輸係一種常見而且有禮貌嘅結束方式。
