Cara Menggunakan Situs Weiqi.Kids

Selamat datang di situs kami! Di sini kami menyediakan sumber daya untuk pemain Go dan insinyur AI. Anda dapat mengakses berbagai topik melalui menu di sebelah kiri.

Navigasi Bagian​

  • Untuk Pemain Go: Dibagi berdasarkan tingkat kemampuan menjadi Pemula, Pra-Dan, dan Pasca-Dan.
  • Untuk Insinyur: Menyediakan informasi tentang arsitektur KataGo, latar belakang penelitian, dan kode sumber.
  • Ringkasan Evolusi: Mencakup sejarah perkembangan Go manusia dan perjalanan pertandingan AI.
